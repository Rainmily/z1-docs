# 快速入门

## 5分钟快速上手

### 第一步：账号开通

联系客服（400-103-1003）开通试用账号，获取登录链接和初始密码。

### 第二步：基础设置

1. 登录系统
2. 设置门店信息
3. 配置员工账号
4. 添加商品分类

### 第三步：商品录入

- 单品录入：手动添加商品信息
- 批量导入：使用Excel模板批量导入
- 扫码入库：支持扫码快速录入

### 第四步：开始使用

- 收银：首页快速收银入口
- 库存：查看和管理商品库存
- 会员：办理会员卡

### 常见问题

**Q: 如何添加员工？**
A: 系统设置 → 员工管理 → 添加员工

**Q: 如何设置会员等级？**
A: 会员管理 → 会员等级 → 新增等级

**Q: 如何打印小票？**
A: 系统设置 → 打印设置 → 配置打印机
