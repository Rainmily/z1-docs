# 更新日志

记录掌上乾坤系统的版本更新和功能迭代。

## v3.5.0 (2024年12月)

### 新功能

- **AI智能补货建议**：基于销售历史和库存数据，AI自动生成补货建议
- **会员流失预警**：智能识别高流失风险会员，推送挽留提醒
- **新增数据大屏**：门店经营数据可视化大屏，支持TV投屏

### 功能优化

- 收银页面加载速度提升40%
- 库存预警逻辑优化，减少误报
- 报表导出支持更多格式

### 问题修复

- 修复特定机型扫码失败问题
- 修复会员积分扣除异常问题

---

## v3.4.0 (2024年9月)

### 新功能

- **企业微信集成**：支持企业微信消息推送
- **批量操作优化**：支持批量入库、批量调拨
- **员工打卡功能**：支持GPS定位打卡

### 功能优化

- 移动端界面全新改版
- 搜索功能支持模糊匹配

---

## v3.3.0 (2024年6月)

### 新功能

- **多语言支持**：支持简体中文、繁体中文、英文
- **小程序商城**：配套微商城小程序
- **积分商城**：会员积分兑换平台

### 功能优化

- 权限管理更灵活
- 报表自定义程度更高
