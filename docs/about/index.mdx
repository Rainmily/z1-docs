# 关于我们

## 公司简介

晋城市掌上乾坤网络科技有限公司成立于2014年，专注于手机门店数智化服务10年+。

### 发展历程

| 年份 | 里程碑 |
|------|--------|
| 2014年 | 公司成立，推出第一代门店管理系统 |
| 2016年 | 服务门店突破1000家 |
| 2018年 | 推出移动端APP，支持手机、平板使用 |
| 2020年 | 会员营销模块上线，助力门店私域运营 |
| 2022年 | AI智能分析功能上线，数据驱动决策 |
| 2024年 | 服务门店突破5000家，品牌升级 |

### 核心优势

- **10年专注**：深耕手机零售行业，技术与业务深度融合
- **5000+客户**：服务各规模手机门店，经验丰富
- **专业技术**：稳定的技术团队，持续迭代优化
- **贴心服务**：7×24小时服务热线，及时响应
