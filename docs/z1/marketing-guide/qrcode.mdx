# 签码管理

## 业务场景

签码管理用于管理门店二维码，支持扫码签到、扫码支付等场景。

## 前置条件

- 二维码已生成

## 名词解释

| 术语 | 说明 |
|------|------|
| 签码 | 扫码签到的二维码 |
| 支付码 | 扫码支付的二维码 |

## 操作步骤

### 生成二维码

1. 进入 **营销管理** → **签码管理**
2. 点击 **生成二维码**
3. 选择类型
4. 下载使用

### 查看统计数据

1. 查看扫码记录
2. 统计使用情况
