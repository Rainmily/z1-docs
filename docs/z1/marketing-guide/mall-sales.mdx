# 商城销售服务

## 业务场景

商城销售服务用于处理小程序商城的订单和售后服务，包括订单处理、发货管理、退换货处理等全流程业务。

## 前置条件

- 商城已开通
- 有订单产生
- 具有相应的订单管理权限

## 订单类型

| 类型 | 说明 |
|------|------|
| 普通订单 | 标准商品购买 |
| 预售订单 | 预订商品订单 |
| 换新订单 | 以旧换新订单 |

## 操作步骤

### 处理订单

1. 进入 **商城管理** → **订单管理**
2. 查看所有订单列表
3. 筛选订单状态（待付款/待发货/已完成等）
4. 点击订单查看详情
5. 进行相应操作

### 订单操作

| 操作 | 说明 |
|------|------|
| 确认订单 | 确认订单信息 |
| 发货 | 为已付款订单发货 |
| 查看物流 | 跟踪物流状态 |
| 退款 | 处理退款申请 |
| 取消 | 取消订单 |

### 售后服务

1. 进入 **售后服务** 页面
2. 查看退换货申请列表
3. 点击申请查看详情
4. 审核通过/驳回申请
5. 执行退款或退货操作

### 退货退款

1. 审核通过退货申请
2. 等待顾客退货
3. 收到退货后确认
4. 完成退款

## 常见问题

### 订单超时未付款怎么办？

订单24小时未付款自动取消。

### 可以部分发货吗？

可以，支持拆单发货。

### 退款多久到账？

原路退款1-7个工作日到账。
