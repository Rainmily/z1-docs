# 自定义商城活动页

## 业务场景

自定义商城活动页用于创建专属营销活动页面，支持配置活动链接、商品、卡券等，满足线上营销需求，支持地推商户拜访登记与发券。

## 前置条件

- 已配置活动页面素材
- 参与活动的商品/卡券已创建

## 名词解释

| 术语 | 说明 |
|------|------|
| 活动PV | 页面访问量 |
| 总邀请人数 | 通过邀请码访问的用户数 |
| 活动热区 | 活动图片上的可点击区域 |
| 邀请码 | 区分访问渠道的专属码 |

## 活动属性

| 属性 | 说明 |
|------|------|
| 活动名称 | 20字以内，不可重复 |
| 活动时间 | 开始/结束时间 |
| 活动状态 | 根据时间和开关自动计算 |
| 活动图片 | 支持 JPG/PNG/GIF |
| 参与部门 | 多选，部门白名单 |

## 热区动作

| 动作类型 | 说明 |
|----------|------|
| 页面路径 | 小程序内部页面链接 |
| 链接商品 | 已上架商品 SPU |
| 预订活动 | 已发布的预订活动 |
| 秒杀活动 | 启用状态的预订活动 |
| 绑定卡券 | 领取规则关联的卡券 |
| 链接表单 | 已发布的表单 |
| 联系方式 | 手机号 |

## 操作步骤

### 创建活动

1. 进入 **营销管理** → **商城活动**
2. 点击 **新建活动**
3. 填写活动名称、时间
4. 上传活动图片
5. 配置活动热区
6. 设置参与部门
7. 保存

### 配置热区

1. 在活动图片上框选区域
2. 设置热区名称
3. 选择热区动作
4. 保存热区

### 邀请设置

1. 开启邀请开关
2. 生成职员邀请码
3. 生成部门邀请码
4. 分发给相关人员

### 活动管理

| 操作 | 说明 |
|------|------|
| 启用 | 活动可被访问 |
| 停用 | 页面无法访问（停用后可编辑） |

## 常见问题

### 邀请码有什么区别？

- 职员邀请码：记录职员信息
- 部门邀请码：记录部门信息

### 活动可以修改吗？

停用后可以编辑活动页面。

### 非参与部门可以访问吗？

不能，非白名单部门无法访问。
