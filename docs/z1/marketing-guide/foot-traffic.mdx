# 门店客流统计

## 业务场景

门店客流统计用于统计门店进出人数，分析客流趋势。

## 前置条件

- 客流设备已对接

## 名词解释

| 术语 | 说明 |
|------|------|
| 进店人数 | 进入门店的人数 |
| 客流转化率 | 进店人数/下单人数 |

## 操作步骤

### 查看统计数据

1. 进入 **营销管理** → **客流统计**
2. 选择门店和时间
3. 查看统计数据

### 数据分析

1. 分析客流高峰时段
2. 分析转化率
3. 优化经营策略
