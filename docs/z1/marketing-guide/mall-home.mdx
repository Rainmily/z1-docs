# 商城首页设置

## 业务场景

商城首页设置用于配置小程序商城首页的展示内容。

## 前置条件

- 商城已开通

## 设置内容

| 模块 | 说明 |
|------|------|
| Banner | 首页轮播图 |
| 分类入口 | 商品分类快捷入口 |
| 活动专区 | 营销活动入口 |
| 推荐商品 | 商品推荐展示 |

## 操作步骤

### 配置首页

1. 进入 **商城管理** → **首页设置**
2. 上传Banner图
3. 配置分类入口
4. 设置活动专区
5. 保存
