# 卡券

## 业务场景

卡券是系统内各类优惠券、代金券的统称，用于促销活动和客户回馈。卡券可以减免毛利或抵扣金额，帮助门店提升销售转化率。

## 卡券类型

| 类型 | 科目 | 使用位置 | 可叠加 | 适用订单 |
|------|------|----------|--------|----------|
| 通用优惠券 | 否 | 商品 | 否 | 商城订单 |
| 维修优惠券 | 否 | 商品 | 否 | 维修订单 |
| 换新补贴 | 否 | 商品 | 否 | 商城订单（需关联回收单） |
| 通用代金券 | 是 | 订单 | 是 | 商城订单 |
| 回收加价券 | 是 | 回收单 | 是 | 回收订单 |
| 售后权益卡 | 是 | 订单 | 否 | 商城订单 |

## 券类型说明

| 类型 | 说明 |
|------|------|
| 优惠券减免 | 减少毛利，不记账 |
| 代金券抵扣 | 公司承担面额，需要记账 |

## 操作步骤

### 创建卡券

1. 进入 **营销管理** → **卡券管理**
2. 点击 **新建卡券**
3. 选择卡券类型
4. 设置卡券名称、有效期、面值等
5. 保存卡券

### 发放卡券

1. 选择要发放的卡券
2. 选择发放对象（会员/导购）
3. 设置发放数量
4. 确认发放

### 使用卡券

#### 优惠券使用

1. 顾客下单时
2. 系统自动显示可用优惠券
3. 选择使用优惠券
4. 系统自动计算减免金额

#### 代金券使用

1. 顾客下单时
2. 输入代金券码
3. 系统验证代金券有效性
4. 抵扣相应金额

## 叠加规则

| 券类型 | 叠加规则 |
|--------|----------|
| 通用优惠券 | 不可叠加，不可与同类叠加 |
| 通用代金券 | 可与优惠券叠加使用 |
| 换新补贴 | 需关联回收单使用 |

## 常见问题

### 优惠券和代金券有什么区别？

优惠券减免毛利不记账，代金券抵扣金额需要记账。

### 卡券可以转让吗？

部分卡券支持转赠，具体见卡券设置。

### 卡券过期了怎么办？

过期后自动失效，不可使用。
