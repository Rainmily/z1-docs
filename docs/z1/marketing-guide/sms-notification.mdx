# 短信通知

## 业务场景

短信通知用于向会员发送各类营销短信，包括生日祝福、活动通知、促销信息、订单通知等，是会员触达的重要渠道之一。

## 前置条件

- 已开通短信服务
- 短信签名已认证
- 会员手机号已录入

## 名词解释

| 术语 | 说明 |
|------|------|
| 短信签名 | 短信发送者的标识 |
| 短信模板 | 预设的短信内容格式 |
| 短信签名 | 发送短信前需配置签名 |

## 短信类型

| 类型 | 说明 |
|------|------|
| 生日祝福 | 会员生日提醒 |
| 活动通知 | 营销活动推送 |
| 促销信息 | 优惠活动告知 |
| 订单通知 | 订单状态变更提醒 |
| 会员关怀 | 节日问候等 |

## 操作步骤

### 配置短信签名

1. 进入 **系统设置** → **短信设置**
2. 添加短信签名
3. 等待审核认证

### 创建短信模板

1. 点击 **短信模板**
2. 新建模板
3. 编写短信内容
4. 提交审核

### 发送短信

1. 进入 **营销管理** → **短信群发**
2. 选择发送对象
3. 选择短信模板
4. 设置发送时间
5. 确认发送

### 发送记录

1. 查看发送历史
2. 查看发送状态
3. 查看送达率

## 常见问题

### 短信发送失败怎么办？

1. 检查手机号格式是否正确
2. 检查短信签名是否认证
3. 检查模板是否审核通过

### 单次发送有数量限制吗？

有，具体限制根据短信套餐确定。

### 如何退订短信？

短信中包含退订回T，用户回复T即可退订。
