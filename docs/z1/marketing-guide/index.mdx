# 营销模块

营销模块提供多种营销工具，支持卡券发放、活动创建、客流统计等功能，帮助门店提升销售业绩。

## 营销工具

| 功能 | 说明 |
|------|------|
| [卡券](./coupon) | 卡券发放和使用管理 |
| [卡券转赠](./coupon-gift) | 卡券转赠好友 |
| [自定义活动页](./custom-page) | 创建自定义营销活动页 |
| [门店客流统计](./foot-traffic) | 统计门店客流情况 |
| [签码管理](./qrcode) | 二维码签到管理 |
| [短信通知](./sms-notification) | 短信群发通知 |
