# 预售活动

## 业务场景

预售活动用于在商品正式发售前进行预订销售，锁定客户。

## 前置条件

- 预售商品已配置
- 预订名额已设置

## 名词解释

| 术语 | 说明 |
|------|------|
| 预订价 | 预订时的优惠价格 |
| 尾款 | 发售后的剩余款项 |

## 操作步骤

### 创建预售活动

1. 进入 **营销管理** → **预售活动**
2. 点击 **新建活动**
3. 设置商品和价格
4. 设置预订名额
5. 保存

### 管理预售订单

1. 查看预订订单
2. 催付尾款
3. 完成发售
