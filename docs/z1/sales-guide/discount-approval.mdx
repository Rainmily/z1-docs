# 优惠审批流程

## 业务场景

优惠审批用于管理商品改价流程。当商品通过改价、赠品、优惠券等方式优惠后金额低于大盘价时，需要提交审批，确保价格合规。

## 前置条件

- 职员具有改价权限
- 订单状态为"未支付"

## 审批触发条件

| 优惠方式 | 说明 |
|----------|------|
| 改价 | 修改商品售价低于大盘价 |
| 赠品 | 添加赠品后影响宿主商品利润 |
| 优惠券 | 使用优惠券使商品价格低于大盘价 |

## 审批内容

| 字段 | 说明 |
|------|------|
| 订单编号 | 关联的销售订单号 |
| 改价商品 | 享受优惠的商品 |
| 数量 | 优惠商品数量 |
| 改价前售价 | 系统标准零售价 |
| 改价后售价 | 调整后的售价 |
| 改价总金额 | 优惠金额 × 数量 |
| 大盘价 | 商品标准成本价 |

## 审批流程

### 提交申请

1. 创建/编辑订单
2. 进行改价/赠品/优惠券操作
3. 系统判断金额低于大盘价
4. 自动跳转到审批提交页面
5. 确认信息并提交

### 审批层级

| 改价总金额 | 审批流程 |
|------------|----------|
| ≤ 200元 | 店长审批 |
| > 200元 | 采销商务审批 |

### 钉钉审批

- 审批通过后，系统自动完成改价
- 审批拒绝后，单据消失，需重新操作

## 订单状态

| 状态 | 说明 |
|------|------|
| 优惠审批中 | 等待审批，无法继续操作 |
| 已完成 | 审批通过，可继续流程 |

## 操作步骤

### 查看待审批列表

1. 进入 **审批管理** → **优惠审批**
2. 查看待审核单据

### 查看已审批记录

1. 支持按日期、部门、职员、订单号筛选
2. 查看已通过的单据

## 常见问题

### 审批通过后可以退款吗？

可以正常退款。

### 订单已支付还能改价吗？

不能，订单状态为"已支付"时不允许改价。

### 审批拒绝怎么办？

需要重新进行优惠操作并提交审批。
