# 赠品功能

## 业务场景

赠品是一种促销手段，通过免费赠送附加商品或服务吸引消费者，提升购买欲望和销售额。赠品不改变主商品价格，避免价格竞争的同时达到促销效果。

## 前置条件

- 已在系统设置中配置赠品可选范围
- 商品库存充足

## 名词解释

| 术语 | 说明 |
|------|------|
| 赠品 | 免费给消费者的附加商品或服务 |
| 赠品宿主 | 赠品所绑定的购买商品 |
| 手动赠品 | 职员下单时手动添加的赠品 |
| 默认赠品 | 系统根据赠品方案自动匹配的赠品 |
| 大盘价 | 商品的标准成本价 |

## 赠品规则

### 基本规则

| 规则 | 说明 |
|------|------|
| 赠品不可为宿主本身 | 赠品必须是其他商品 |
| 赠品不可再绑赠品 | 赠品不能作为其他赠品的宿主 |
| 赠品不参与优惠 | 不计入折扣、优惠券等优惠 |

### 数量规则

```
赠品数量 = 赠品宿主数量 × n（n≥1）
```

每个宿主商品可关联多个赠品，数量按比例配置。

### 价格规则

| 界面 | 赠品价格 |
|------|----------|
| 顾客界面 | 0元 |
| 职员界面 | 成本价 |

### 优惠叠加

改价、赠品、优惠券可以同时叠加，但最终折扣后金额不能低于大盘价。

## 赠品类型

### 手动赠品

职员下单时手动添加：
- 可选商品或服务
- 受库存限制
- 可覆盖默认赠品
- 受最大赠品数量限制

### 默认赠品

提前配置赠品方案：
- 系统根据购买数量自动匹配
- 下单时可转为手动赠品

## 操作步骤

### 配置手动赠品范围

1. 进入 **系统设置** → **手动赠品设置**
2. 设置可选SKU和服务范围
3. 保存配置

### 下单添加赠品

1. 创建/编辑销售订单
2. 选择商品后点击"添加赠品"
3. 选择赠品宿主商品
4. 从可选范围中选择赠品
5. 确认赠品信息

### 配置默认赠品方案

1. 进入 **营销管理** → **赠品方案**
2. 创建新方案
3. 设置赠品宿主商品
4. 配置赠品数量和规则
5. 保存方案

### 查看赠品订单

订单详情中可查看：
- 赠品宿主商品
- 赠品明细
- 赠品成本价

## 常见问题

### 为什么赠品价格显示为0？

顾客界面显示0元，职员界面显示成本价，这是系统设计。

### 赠品可以退换吗？

赠品作为促销商品，一般不支持退换。

### 赠品需要验证大盘价吗？

手动优惠行为时需要验证，默认赠品在出库时不验证。
