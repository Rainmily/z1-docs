# 退货退款

## 业务场景

退货退款功能支持线上线下各种场景的退款退货需求，包括仅退款和退货退款两种模式，保障消费者权益的同时规范退换货流程。

## 前置条件

- 订单状态符合退款条件
- 在退货退款时效内（系统设置天数）
- 商品符合退货要求

## 退款类型

### 仅退款

**适用条件**：订单已支付但未发货

| 说明 | 内容 |
|------|------|
| 退款范围 | 整单退款 |
| 退款金额 | 全额退款 |
| 库存释放 | 自动释放锁货 |
| 卡券处理 | 积分退还，优惠券不退 |

### 退货退款

**适用条件**：订单已完成

| 说明 | 内容 |
|------|------|
| 退款范围 | 支持部分退款 |
| 退货要求 | 赠品需随宿主一起退 |
| 退款金额 | 可修改（不超过原商品金额） |
| 积分处理 | 扣除已获积分 |

## 退款状态

| 状态 | 说明 |
|------|------|
| 待审核 | 顾客申请退款，等待职员处理 |
| 已审核 | 职员已通过，创建退货单 |
| 已驳回 | 职员驳回，可重新申请 |
| 已退款 | 退款完成 |
| 已取消 | 顾客主动取消 |

## 操作步骤

### 顾客申请仅退款

1. 进入订单详情
2. 点击"申请退款"
3. 选择退款原因
4. 提交申请

### 顾客申请退货退款

1. 进入订单详情
2. 点击"申请退货"
3. 选择退货商品
4. 填写退货原因
5. 上传凭证（如需）
6. 提交申请
7. 等待职员审核

### 职员处理退款申请

1. 进入退款申请列表
2. 查看申请详情
3. 审核通过/驳回
4. 如通过，创建退货单

### 职员手动创建退货

1. 打开销售订单
2. 点击"退货退款"
3. 选择退货商品
4. 确认退款金额
5. 选择退款支付方式
6. 完成退货

## 退款金额计算

### 优惠券处理

- 优惠券金额算在订单商品上
- 按优惠后金额退款
- **优惠券不退**

### 代金券处理

| 情况 | 处理方式 |
|------|----------|
| 退款金额 > 代金券 | 只退超出部分 |
| 退款金额 ≤ 代金券 | 不退现金 |

### 积分处理

- 积分支付部分退还用户积分
- 退货扣除已获积分

## 赠品退货规则

| 规则 | 说明 |
|------|------|
| 宿主退货 | 赠品必须随宿主一起退 |
| 赠品遗失 | 需先退库再报损 |
| 服务含卡券 | 已使用需扣减退款金额 |

## 常见问题

### 退货有时效吗？

有，从订单完成开始计算，具体天数在系统设置中配置。

### 部分退款支持吗？

仅退款不支持部分退款，退货退款支持部分商品退货。

### 退款多久到账？

原路退款1-7个工作日到账，现金当场退还。
