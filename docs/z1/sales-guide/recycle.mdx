# 回收模块

## 业务场景

回收模块用于管理二手手机回收业务，对接机大侠平台，记录回收品从入库到售出的完整过程，支持门店回收和平台转售。

## 前置条件

- 已对接机大侠平台
- 回收规则已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 回收单 | 记录回收品完整过程的单据 |
| 评估价 | 机大侠的门店预估价 |
| 回收报价 | 扣除利润空间后的价格 |
| 门店红包 | 给顾客的回收红包 |
| 渠道 | 二手货品售出方式（企业对企业） |

## 价格计算

```
回收报价 = 市场估价 × (1 - 利润空间率)
最终回收价 = 回收报价 + 门店红包
门店佣金 = 市场估价 - 回收报价
```

## 操作步骤

### 创建回收单

1. 进入 **回收管理** → **回收单**
2. 点击 **新建回收单**
3. 录入商品信息
4. 确认回收价格
5. 完成回收

### 复检操作

1. 收到实物后进行复检
2. 与初次检查报告对比
3. 修改确认最终评估

### 回收结算

1. 完成付款操作
2. 系统自动计算佣金
3. 门店红包发放给顾客

## 常见问题

### 门店红包是什么？

爱回收后台设置的，职员回收时可选择给顾客的回收红包。

### 转售渠道有哪些？

主要是企业对企业平台，如回收宝、拍机堂等。
