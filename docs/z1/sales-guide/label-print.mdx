# 标签打印

## 业务场景

标签打印用于打印商品标签、回收标签、价格标签等，支持二维码和条形码。方便商品管理和库存流转。

## 前置条件

- 打印机已配置
- 标签模板已设置
- 商品信息已录入

## 标签类型

| 类型 | 说明 |
|------|------|
| 商品标签 | 商品基本信息的标签 |
| 价格标签 | 含价格的标签 |
| 回收标签 | 回收商品的专属标签 |
| 序列号标签 | 含序列号的标签 |

## 支持格式

| 格式 | 说明 |
|------|------|
| 条形码 | EAN/UPC码 |
| 二维码 | QR Code |
|流水号 | 自动编号 |

## 操作步骤

### 打印商品标签

1. 进入 **库存管理** → **标签打印**
2. 选择要打印的商品
3. 选择标签模板
4. 设置打印数量
5. 点击打印

### 批量打印

1. 选择多个商品
2. 确认打印清单
3. 选择模板
4. 批量打印

### 自定义标签模板

1. 进入 **系统设置** → **标签模板**
2. 点击 **新建模板**
3. 设计标签布局
4. 添加字段（商品名称、价格、条码等）
5. 设置字体大小
6. 保存模板

## 打印机配置

| 配置项 | 说明 |
|--------|------|
| 打印机名称 | 选择打印机 |
| 纸张尺寸 | 标签纸尺寸 |
| 打印浓度 | 打印深浅 |

## 常见问题

### 打印不清晰怎么办？

1. 调整打印浓度
2. 更换碳带
3. 清洁打印头

### 标签纸用完了怎么办？

联系采购更换标签纸，注意规格匹配。
