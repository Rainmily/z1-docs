# 平台转售管理

## 业务场景

平台转售管理用于将回收的二手商品转售到第三方平台（如回收宝、拍机堂等），实现库存变现。企业对企业的B2B交易模式。

## 前置条件

- 第三方平台已对接
- 回收商品已入库
- 商品符合平台收购条件

## 名词解释

| 术语 | 说明 |
|------|------|
| 转售平台 | 二手商品交易平台 |
| 转售价 | 平台的收购价格 |
| 渠道费 | 平台收取的服务费用 |

## 转售流程

```
商品入库 → 选择平台 → 提交转售 → 平台审核 → 平台打款 → 完成转售
```

## 操作步骤

### 发起转售

1. 进入 **回收管理** → **平台转售**
2. 选择要转售的商品
3. 选择目标平台
4. 确认转售价格
5. 提交转售申请

### 转售状态

| 状态 | 说明 |
|------|------|
| 待审核 | 等待平台审核 |
| 已通过 | 平台审核通过 |
| 已打款 | 款项已到账 |
| 已完成 | 转售完成 |
| 已拒绝 | 平台拒绝收购 |

### 结算对账

1. 查看转售记录
2. 核对平台打款金额
3. 生成财务凭证

## 常见问题

### 转售价格如何确定？

根据商品成色、型号等因素，由平台估价确定。

### 转售失败怎么办？

商品退回仓库，可重新发起转售或转为其他处理方式。

### 平台费用如何结算？

平台自动扣除渠道费，结算金额为转售价减去渠道费。
