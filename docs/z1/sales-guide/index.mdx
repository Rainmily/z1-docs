# 销售模块

销售模块管理门店零售业务，支持多种订单类型和支付方式，帮助门店高效完成销售工作。

## 功能列表

| 功能 | 说明 |
|------|------|
| [销售订单](./sales-order) | 管理零售销售业务 |
| [预订订单](./reservation-order) | 管理预订销售业务 |
| [赠品功能](./gift-function) | 赠品发放和管理 |
| [退货退款](./return-refund) | 处理退货退款业务 |
| [以旧换新](./trade-in) | 旧机换新业务 |
| [优惠审批](./discount-approval) | 优惠价格审批流程 |
| [利润分析](./profit-analysis) | 商品利润统计分析 |
| [回收模块](./recycle) | 二手商品回收 |
| [平台转售](./resale) | 平台转售管理 |
| [标签打印](./label-print) | 商品标签打印 |

## 订单类型

| 类型 | 说明 |
|------|------|
| 零售单 | 门店日常零售业务 |
| 网销单 | 小程序商城订单 |
| 批发单 | 批发业务订单 |
