# 商品利润分析

## 业务场景

商品利润分析用于统计一段时间内各 SKU 商品的利润贡献情况，帮助管理者了解哪些商品带来更多净利润与连带净利润，为经营决策提供数据支持。

## 前置条件

- 系统中有销售订单数据
- 具有相应的数据查看权限

## 统计范围

| 数据类型 | 说明 |
|----------|------|
| 订单数据 | 包含该 SKU 的订单 |
| 调价单数据 | 店内/供应商调价单 |
| 调差单数据 | 供应商调差单 |

## 统计维度

### 主营商品

系统定义的主营商品范围：手机、电脑、平板

### 筛选条件

| 条件 | 说明 |
|------|------|
| 商品 SKU | 多选（分类/SPU/SKU） |
| 部门 | 多选，自包含 |
| 日期 | 时间段，默认当天 |
| 销售类型 | 卖/退 |
| 销售模式 | 店内零售/网销/批发 |

## 分析指标

### 主营指标

| 指标 | 说明 |
|------|------|
| 主营销量 | 主营商品销量 |
| 主营零售总额 | 标准零售价格 |
| 主营成交总额 | 顾客实付金额 |
| 主营改价总额 | 改价造成的损失 |
| 主营毛利总额 | 成交额 - 成本 |

### 连带指标

| 指标 | 说明 |
|------|------|
| 连带销量 | 连带商品销量 |
| 连带零售总额 | 连带商品零售额 |
| 连带成交总额 | 连带商品实付额 |
| 连带毛利总额 | 连带商品毛利 |

## 计算规则

### 退货处理

统计时默认扣除退货商品数据（可选择包含）。

### 订单类型

- 零售单
- 网销单
- 批发单

### 连带计算

一个订单内有多个主营商品时，连带商品需对每个主营商品分别统计一遍。

## 操作步骤

### 查看利润分析

1. 进入 **销售管理** → **商品利润分析**
2. 选择筛选条件
3. 点击查询
4. 查看分析结果

### 导出报表

1. 查看分析结果后
2. 点击导出按钮
3. 选择导出格式
4. 保存报表

## 常见问题

### 为什么利润数据不包含房租水电？

目前系统仅统计商品直接相关的成本，房租、水电等运营成本未列入。

### 退货订单如何处理？

默认在统计时将退货商品减除，可在筛选中选择包含。

### 提成如何计算？

人工提成按商品提成部分预计算，订单提成部分按占比等分。
