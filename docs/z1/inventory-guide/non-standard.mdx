# 非标模块

## 业务场景

非标模块用于管理非标准商品，如二手手机、翻新机等特殊商品的进销存管理。

## 前置条件

- 非标商品分类已配置
- 鉴定标准已建立

## 名词解释

| 术语 | 说明 |
|------|------|
| 非标商品 | 非标准规格的商品 |
| 鉴定等级 | 商品成色等级 |

## 操作步骤

### 非标商品入库

1. 商品鉴定
2. 录入成色等级
3. 定价
4. 入库

### 非标商品销售

1. 选择非标商品
2. 确认销售价格
3. 完成订单
