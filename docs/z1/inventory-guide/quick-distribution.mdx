# 快捷分货

## 业务场景

快捷分货用于快速将商品分配到不同门店或仓库，提高分货效率。适用于商品调配、供货补货等场景，支持批量处理和即时库存更新。

## 前置条件

- 门店/仓库已配置
- 商品已入库
- 有分货权限

## 分货模式

| 模式 | 说明 |
|------|------|
| 门店调拨 | 门店之间的商品调配 |
| 仓库调拨 | 仓库之间的商品调拨 |
| 仓库到门店 | 仓库向门店供货 |

## 操作步骤

### 创建分货任务

1. 进入 **库存管理** → **快捷分货**
2. 选择分货模式
3. 选择要分货的商品
4. 选择分货目标（门店/仓库）
5. 设置分货数量
6. 确认分货

### 批量分货

1. 导入分货清单
2. 系统自动匹配商品
3. 确认分货信息
4. 批量提交

### 分货确认

1. 分货接收方收到通知
2. 确认收到商品
3. 系统更新库存

## 常见问题

### 分货后库存如何变化？

分出货源减少，分入方增加。

### 可以分货到多个门店吗？

可以，支持批量分发到多个门店。

### 分货有数量限制吗？

受分货源库存数量限制。
