# 商品管理

## 业务场景

商品管理用于管理门店销售的商品信息，包括商品上架、价格调整等。

## 前置条件

- 商品分类已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 商品名称 | 商品显示名称 |
| 商品编码 | 商品唯一标识 |
| 零售价 | 销售价格 |
| 成本价 | 进货价格 |

## 操作步骤

### 新增商品

1. 进入 **商品管理** → **商品列表**
2. 点击 **新增商品**
3. 填写商品信息
4. 设置价格
5. 保存

### 商品上下架

1. 选择商品
2. 点击 上架/下架
3. 确认操作
