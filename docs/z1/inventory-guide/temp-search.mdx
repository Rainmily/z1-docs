# 临时找货功能

## 业务场景

临时找货用于紧急查找所需商品库存，支持跨门店调货。当某个门店缺少库存时，可以通过此功能快速查找其他门店的库存情况，并发起调货申请。

## 前置条件

- 门店库存已同步
- 有调货权限

## 名词解释

| 术语 | 说明 |
|------|------|
| 临时找货 | 紧急查找商品库存的行为 |
| 调货申请 | 向有货门店发起的调货请求 |

## 操作步骤

### 发起找货

1. 进入 **采购管理** → **临时找货**
2. 输入要查找的商品名称或编码
3. 系统自动搜索各门店库存
4. 查看库存分布情况

### 申请调货

1. 选择有库存的门店
2. 点击 **申请调货**
3. 填写调货数量
4. 确认提交
5. 等待对方门店确认

### 调货确认

1. 收到调货申请通知
2. 查看申请详情
3. 确认/拒绝调货
4. 如确认，进行发货操作

## 常见问题

### 为什么找不到商品？

1. 商品可能不在库存同步范围内
2. 商品编码输入错误
3. 商品未在系统中录入

### 调货需要付费吗？

调货是内部库存调配，通常不涉及费用。

### 调货时限是多少？

根据各门店实际情况确定，一般当天或次日完成。
