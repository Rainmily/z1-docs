# 返利与价保

## 业务场景

返利与价保用于保护经销商利益，在商品降价时获得差价补偿。

## 前置条件

- 供应商已配置价保政策

## 名词解释

| 术语 | 说明 |
|------|------|
| 价保 | 价格保护承诺 |
| 差价补偿 | 降价时返还差价 |

## 操作步骤

### 申请价保

1. 发现商品降价
2. 进入 **采购管理** → **价保申请**
3. 填写降价信息
4. 提交申请

### 审核与补偿

1. 供应商审核
2. 确认差价金额
3. 获得补偿
