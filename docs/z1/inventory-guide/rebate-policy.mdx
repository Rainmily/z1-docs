# 返利政策

## 业务场景

返利政策用于设置供应商返利规则，激励采购进货。

## 前置条件

- 供应商信息已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 返利比例 | 采购金额返利百分比 |
| 返利门槛 | 触发返利的条件 |

## 操作步骤

### 配置返利政策

1. 进入 **采购管理** → **返利政策**
2. 点击 **新建政策**
3. 选择供应商
4. 设置返利规则
5. 保存

### 返利计算

1. 系统自动计算返利
2. 生成返利单据
3. 抵冲采购款项
