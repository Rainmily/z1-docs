# 商品SPU/SKU

## 业务场景

SPU是商品标准化产品单元，SKU是库存量单位。用于精细化管理商品规格。

## 前置条件

无

## 名词解释

| 术语 | 说明 |
|------|------|
| SPU | 标准化产品单元（如iPhone 15） |
| SKU | 具体规格（如iPhone 15 黑色 128G） |

## 操作步骤

### 创建SPU

1. 进入 **商品管理** → **SPU管理**
2. 点击 **新建SPU**
3. 填写商品信息
4. 保存

### 添加SKU

1. 选择SPU
2. 点击 **添加SKU**
3. 设置规格和价格
4. 保存
