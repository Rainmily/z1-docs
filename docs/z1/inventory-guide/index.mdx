# 库存模块

库存模块管理门店商品库存，支持调拨、维修、序列号追踪等功能，帮助门店精准掌握库存状况。

## 功能列表

| 功能 | 说明 |
|------|------|
| [调拨单](./transfer-order) | 管理门店之间商品调拨 |
| [返厂维修](./repair-return) | 处理商品返厂维修业务 |
| [序列号追踪](./serial-trace) | 追踪商品序列号流向 |
| [快捷分货](./quick-distribution) | 快速分配商品到各门店 |
| [智能库存](./smart-inventory) | 智能补货和调拨建议 |
| [非标模块](./non-standard) | 处理非标准商品 |
| [商品管理](./product-management) | 管理商品信息 |
| [商品SPU/SKU](./spu-sku) | 管理SPU和SKU规格 |
| [返利政策](./rebate-policy) | 供应商返利政策 |
| [返利与价保](./rebate-price-protect) | 返利和价格保护 |
| [采购功能](./purchase) | 采购订单管理 |
| [临时找货](./temp-search) | 紧急查找商品库存 |
| [太力订单](./taili) | 对接太力订单系统 |
| [报货单](./replenish) | 商品报货管理 |
