# 返厂维修

## 业务场景

返厂维修用于管理需要返回厂家维修的商品，跟踪维修进度和结果。

## 前置条件

- 供应商信息已配置
- 商品符合返厂条件

## 名词解释

| 术语 | 说明 |
|------|------|
| 返厂单 | 商品返厂维修的单据 |
| 维修状态 | 当前维修进度 |

## 操作步骤

### 创建返厂单

1. 进入 **库存管理** → **返厂维修**
2. 点击 **新建返厂单**
3. 选择商品和供应商
4. 填写故障信息
5. 提交

### 跟踪维修进度

1. 查看返厂单状态
2. 跟进维修进度
3. 维修完成后入库
