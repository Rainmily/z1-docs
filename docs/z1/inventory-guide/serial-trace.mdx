# 序列号追踪

## 业务场景

序列号追踪用于跟踪商品序列号的流转记录，实现商品全生命周期管理。

## 前置条件

- 商品已绑定序列号
- 序列号已录入系统

## 名词解释

| 术语 | 说明 |
|------|------|
| 序列号 | 商品的唯一标识 |
| 流转记录 | 序列号的所有操作历史 |

## 操作步骤

### 查询序列号

1. 进入 **库存管理** → **序列号追踪**
2. 输入序列号
3. 查看流转记录

### 序列号状态

| 状态 | 说明 |
|------|------|
| 库存 | 在仓库中 |
| 已销售 | 已售出 |
| 维修中 | 返厂维修 |
| 已注销 | 已销毁 |
