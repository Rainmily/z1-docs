# 太力订单对接

## 业务场景

太力订单对接用于与太力供应链平台对接，自动同步订单和库存。

## 前置条件

- 太力账户已配置
- API已对接

## 功能说明

| 功能 | 说明 |
|------|------|
| 订单同步 | 自动同步采购订单 |
| 库存同步 | 实时更新库存数据 |
| 物流跟踪 | 订单物流状态 |

## 操作步骤

### 配置对接

1. 进入 **系统设置** → **第三方对接**
2. 配置太力API
3. 测试连接

### 查看同步数据

1. 查看订单同步状态
2. 查看库存同步记录
