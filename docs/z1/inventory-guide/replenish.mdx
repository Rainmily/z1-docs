# 报货单

## 业务场景

报货单是门店向仓库或供应商申请补货的单据。

## 前置条件

- 仓库已配置
- 供应商已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 报货单 | 申请补货的单据 |
| 报货数量 | 申请补货的数量 |

## 操作步骤

### 创建报货单

1. 进入 **采购管理** → **报货单**
2. 点击 **新建报货单**
3. 选择商品
4. 填写数量
5. 提交

### 审核与发货

1. 仓库审核
2. 确认发货
3. 商品入库
