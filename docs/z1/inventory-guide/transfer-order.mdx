# 调拨单

## 业务场景

调拨单用于管理门店之间的商品调拨业务，实现库存的合理分配和转移。

## 前置条件

- 调出仓库和调入仓库已配置
- 有可调拨的库存

## 名词解释

| 术语 | 说明 |
|------|------|
| 调拨单 | 商品在不同仓库间转移的单据 |
| 调出仓库 | 商品发出的仓库 |
| 调入仓库 | 商品接收的仓库 |

## 操作步骤

### 创建调拨单

1. 进入 **库存管理** → **调拨单**
2. 点击 **新建调拨单**
3. 选择调出/调入仓库
4. 添加调拨商品
5. 确认数量
6. 提交审核

### 审核调拨

1. 审核通过后
2. 调出仓库减少库存
3. 调入仓库增加库存
