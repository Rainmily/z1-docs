# 储值功能操作指南

## 业务场景

储值功能是提升顾客粘性、增加复购率的有效工具。通过让顾客预存金额，可以提前锁定消费，同时通过赠送机制刺激顾客充值。

**适用场景：**
- 顾客单次消费金额较大，希望分次使用
- 门店促销活动，如"充100送20"
- 建立会员忠诚计划，鼓励重复消费

## 前置条件

- 需要有权限的职员账号
- 系统已开启储值功能配置
- 顾客已在系统中注册为会员

## 名词解释

| 术语 | 说明 |
|------|------|
| 储值 | 顾客预存金额到虚拟账户 |
| 预存本金 | 顾客实际支付的金额，可退款 |
| 赠送金额 | 平台赠送的金额，不可退款 |
| 储值订单 | 记录储值和退款的文件 |

## 操作步骤

### 步骤1：进入储值模块

在系统菜单中选择 **会员模块** → **储值功能**

### 步骤2：输入顾客信息

1. 输入顾客的手机号码
2. 点击查询，系统显示顾客信息和当前储值余额

### 步骤3：选择储值金额

1. 选择预设金额（系统提供快捷选项）
2. 或手动输入自定义金额
3. 确认后点击"马上储值"

### 步骤4：完成支付

1. 选择支付方式（现金/银行卡/微信/支付宝）
2. 完成支付后，系统自动创建储值订单
3. 储值金额即时到账

## 储值退款操作

### 退款规则

- **预存本金**：可退款，按原支付路径返回
- **赠送金额**：不可退款
- 支持多次部分退款

### 退款步骤

1. 进入 **储值订单** 列表
2. 找到需要退款的订单
3. 点击"申请退款"
4. 输入退款金额（不超过剩余本金）
5. 确认退款

## 常见问题

### Q: 储值后可以提现吗？

A: 不可以。储值金额只能在门店消费使用，不能提现。

### Q: 赠送的金额怎么使用？

A: 赠送金额与本金合并使用，消费时按比例抵扣。退款时赠送金额不退。

### Q: 储值订单多久会自动取消？

A: 创建后30分钟未支付会自动取消。

## 运营技巧

1. **推荐储值金额**：根据顾客消费习惯，推荐合适的储值档位
2. **活动时机**：节假日、店庆时推出储值赠送活动效果更好
3. **储值告知**：向顾客明确说明储值规则和有效期
