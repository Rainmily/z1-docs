# 专属导购

## 业务场景

专属导购功能用于建立会员与导购之间的专属服务关系，实现1v1精准服务，提升客户购物体验和忠诚度。

## 前置条件

- 职员入职后自动生成专属导购信息（默认停用，需店长启用）
- 需要配置导购二维码供客户扫描绑定

## 名词解释

| 术语 | 说明 |
|------|------|
| 专属导购 | 给客户提供1v1专属服务的营业员 |
| 专属客户 | 与专属导购绑定专属关系的会员 |
| 累计拉新客户数 | 从成为专属导购起累计拉新的客户数量 |
| 现有服务客户数 | 现有的专属客户总人数 |
| 转移客户 | 将专属客户转移给其他导购 |

## 绑定规则

| 场景 | 说明 |
|------|------|
| 扫码绑定 | 通过专属导购二维码扫码绑定 |
| 绑定限制 | 每位客户最多绑定2名专属导购 |
| 跨店绑定 | 可在不同门店绑定，不受门店限制 |

### 绑定结果

| 结果 | 条件 |
|------|------|
| 绑定成功 | 客户专属导购数不足2名 |
| 绑定失败 | 客户已有2名专属导购（含2名） |

## 解绑规则

- 客户可自行解除与专属导购的绑定关系
- 专属导购可主动与客户解除绑定关系

## 导购属性

| 属性 | 说明 |
|------|------|
| 导购名称 | 导购姓名 |
| 门店 | 导购所属门店 |
| 工号 | 导购工号 |
| 累计拉新客户数 | 累计拉新客户总数 |
| 现有服务客户数 | 当前专属客户数 |
| 状态 | 启用/停用 |
| 导购码 | 专属二维码 |

## 操作步骤

### 启用专属导购

1. 店长进入 **导购管理**
2. 找到对应职员
3. 将状态设置为"启用"

### 客户绑定专属导购

1. 客户扫描导购专属二维码
2. 系统判断绑定数量
3. 绑定成功/失败提示

### 转移客户

当导购离职或调岗时，店长可将客户转移给其他导购：

1. 选择要转移的导购（源）
2. 选择接收导购（目标）
3. 系统平均分配客户

### 查看我的客户

导购可查看自己的专属客户列表，包括：
- 客户基本信息
- 成为客户的时间
- 累计消费金额
- 会员等级

## 常见问题

### 一位客户能绑定几名专属导购？

最多2名。

### 导购离职后客户怎么办？

店长可将客户转移给其他导购。

### 客户可以跨店绑定导购吗？

可以，客户可在任意门店绑定导购，不受门店限制。
