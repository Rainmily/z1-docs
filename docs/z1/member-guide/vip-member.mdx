# 付费会员

## 业务场景

付费会员是用户通过付费购买才能加入的会员体系，对标"京东Plus"、"淘宝88"等行业标杆模式，为高价值客户提供更优质的专属服务。与普通会员区分，提供专属权益。

## 前置条件

- 已在后台配置付费会员等级（最多5个）
- 会员可通过小程序或门店购买

## 名词解释

| 术语 | 说明 |
|------|------|
| 付费会员 | 需付费购买才能加入的会员体系，有到期时间，到期后自动失效 |
| 等级权重 | 1-10的整数，决定等级优先级 |
| 有效期 | 付费会员的有效期限，到期后自动失效 |

## 付费会员等级配置

每个付费会员等级包含以下配置：

| 配置项 | 说明 |
|--------|------|
| 等级名称 | 自定义名称，10字以内 |
| 等级权重 | 1-10整数，决定等级优先级 |
| 等级标识 | 图片icon，一张 |
| 会员介绍详情 | 多张图片，固定750宽度 |
| 会员权益 | 服务卡券包 + 其他权益文字描述 |
| 标价 | 划线价格，没有则不渲染 |
| 售价 | 实际价格，必须小于标价 |
| 会员条款 | 链接至政策中心相关文章 |
| 专属兑换商品 | 可用积分兑换的商品 |
| 专属专题活动页 | 付费会员专属活动页面 |

### 会员权益

- **服务卡券包**：购买后一次性全部发放，支持优惠券、代金券、售后权益卡
- **其他权益**：文字描述（专属客服、市区内免邮、只换不修等）
- **专属卡券**：配合领取规则，一定周期内可领取的数量限制

### 价格设置

| 类型 | 说明 |
|------|------|
| 标价 | 划线价格，没有则不渲染 |
| 售价 | 实际价格，必须小于标价 |

## 开通与升级

### 开通方式

| 场景 | 操作 |
|------|------|
| 未开通会员 | 直接购买任意等级 |
| 已开通会员 | 可升级更高等级 |

### 升级方式

**直接购买升级并立即生效**：
- 直接购买高等级会员，覆盖原等级（相当于另外开通一次会员）
- 原等级卡券保留、未使用的其他权益作废
- 有效期重置：从当天重新计算（365天）

### 升级计算示例

```
黄金会员：100元/年
白金会员：200元/年

张三1月1日开通黄金会员，有效期至26.1.1
4月1日升级白金会员，此时黄金剩余天数为275天

补差金额 = 目标等级全年价格 - (已支付金额 / 总天数 × 剩余天数)
补差金额 = 200 - (100/365 × 275) ≈ 124.66元
支付124.66元后，立即升级为白金会员，有效期为升级当天+365天
```

### 升级方式对比

| 方式 | 说明 |
|------|------|
| 按剩余价值补差 | 补差金额 = 目标价 - (已支付价/总天数×剩余天数)，全年升级 |
| 按剩余天数补差 | 仅升级剩余天数的权益，有效期不变 |
| 直接购买升级 | 等当前等级到期后生效，有效期延长一年 |

## 会员到期处理

- 到期后已发放未使用的卡券仍可使用（以卡券本身使用规则为准）
- 付费会员不累计普通会员经验值，与普通会员互不影响

## 常见问题

### 付费会员最多能创建几个等级？

最多创建5个付费会员等级。

### 付费会员与普通会员有什么区别？

付费会员需付费购买，有到期时间限制，享受专属权益。普通会员免费注册，通过消费累积经验值升级。

### 会员到期后卡券还能用吗？

已发放未使用的卡券在到期后仍可使用，以卡券本身规则为准。

### 升级后有效期如何计算？

直接购买升级并立即生效：有效期从升级当天重新计算365天。
