# 付费会员

## 业务场景

付费会员是用户通过付费购买才能加入的会员体系，对标"京东Plus"、"淘宝88"等行业标杆模式，为高价值客户提供更优质的专属服务。

## 前置条件

- 已在后台配置付费会员等级
- 会员可通过小程序或门店购买

## 名词解释

| 术语 | 说明 |
|------|------|
| 付费会员 | 需付费购买才能加入的会员体系 |
| 等级权重 | 1-10的整数，决定等级优先级 |
| 有效期 | 付费会员的有效期限，到期后自动失效 |

## 付费会员等级配置

每个付费会员等级包含以下配置：

| 配置项 | 说明 |
|--------|------|
| 等级名称 | 自定义名称，10字以内 |
| 等级权重 | 1-10整数 |
| 等级标识 | 图片icon |
| 会员介绍 | 多张图片，固定750宽度 |
| 会员权益 | 服务卡券包 + 其他权益文字描述 |
| 标价 | 划线价格（可选） |
| 售价 | 实际价格，必须小于标价 |
| 会员条款 | 链接至政策中心 |

### 会员权益

- **服务卡券包**：购买后一次性发放，支持优惠券、代金券、售后权益卡
- **其他权益**：专属客服、市区内免邮、只换不修等
- **专属卡券**：一定周期内可领取数量限制

## 开通与升级

### 开通方式

| 场景 | 操作 |
|------|------|
| 未开通会员 | 直接购买任意等级 |
| 已开通会员 | 可升级更高等级 |

### 升级方式

**直接购买升级（立即生效）**：
- 直接购买高等级会员，覆盖原等级
- 原等级卡券保留，其他权益作废
- 有效期重置为当天+365天

### 升级计算示例

```
黄金会员：100元/年
白金会员：200元/年

张三1月1日开通黄金会员，有效期至次年1月1日
4月1日升级白金会员，剩余275天

补差金额 = 200 - (100/365 × 275) ≈ 124.66元
升级后有效期 = 4月1日 + 365天
```

## 会员到期处理

- 到期后已发放未使用的卡券仍可使用（以卡券规则为准）
- 付费会员不累计普通会员经验值

## 常见问题

### 付费会员最多能创建几个等级？

最多创建5个付费会员等级。

### 付费会员与普通会员有什么区别？

付费会员需付费购买，有到期时间限制，享受专属权益。普通会员免费注册，通过消费累积经验值升级。

### 会员到期后卡券还能用吗？

已发放未使用的卡券在到期后仍可使用，以卡券本身规则为准。
