# 会员总览

## 业务场景

会员总览用于对会员数据进行综合汇总统计，帮助管理者全面了解会员体系运营状况，包括会员人数、人群分布、活跃度、消费情况、复购率等核心指标。

## 前置条件

- 系统中有会员数据
- 具有相应的数据查看权限

## 数据指标

| 指标 | 说明 |
|------|------|
| 会员总人数 | 当前所有会员数量 |
| 已触发会员 | 有消费记录的会员 |
| 未触发会员 | 无消费记录的会员 |
| 本月新增 | 本月新注册的会员数 |
| 本月活跃 | 本月有操作的会员数 |

## 统计维度

| 维度 | 说明 |
|------|------|
| 人群分布 | 按地区、门店、导购等维度统计 |
| 活跃度 | 按最后购买时间、消费频次等分析 |
| 消费分析 | 累计消费金额、消费频次、客单价等 |
| 复购率 | 重复购买会员占比 |

## 操作步骤

### 进入会员总览

1. 进入 **会员管理** → **会员总览**
2. 查看核心数据概览

### 查看详细数据

1. 选择时间范围筛选
2. 选择门店/部门筛选
3. 查看各维度详细统计数据

### 导出报表

点击导出按钮可导出统计数据。

## 常见问题

### 活跃度如何定义？

活跃度根据会员的最后购买时间、登录时间、操作记录等综合计算。

### 复购率如何计算？

复购率 = 有多次消费记录的会员数 / 总会员数 × 100%
