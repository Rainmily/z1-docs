# 会员行为

## 业务场景

会员行为功能用于记录和分析会员在平台上的操作日志与浏览记录，帮助门店了解会员兴趣和需求，为精准营销提供数据支持。

## 前置条件

- 会员需已完成注册（不记录游客行为）
- 系统开启行为数据记录功能

## 名词解释

| 术语 | 说明 |
|------|------|
| 会员行为 | 会员注册后的一系列操作日志与浏览记录 |
| 用户操作详情记录 | 用户ID + 时间 + 页面ID + 事件ID 的组合记录 |

## 记录的行为类型

| 类型 | 说明 |
|------|------|
| 拉新 | 新会员注册 |
| 获取专属导购 | 导购绑定操作 |
| 触发会员等级提升 | 经验值变化 |
| 需求提醒 | 运营商/换机等需求 |
| 会员回访 | 回访记录 |

## 数据统计

### 用户操作日志统计

| 指标 | 说明 |
|------|------|
| 页面ID | 页面唯一标识 |
| 页面访问次数 | 页面被访问的次数 |
| 事件ID | 事件唯一标识 |
| 事件触发次数 | 事件被触发的次数 |

### TOP 排行榜

| 榜单 | 说明 |
|------|------|
| 访问次数最多页面 | 页面热度排行 |
| 收藏次数最多页面 | 会员收藏偏好 |
| 加入购物车次数最多 | 商品热度排行 |

## 常见问题

### 游客行为会被记录吗？

不会，只记录已注册的会员行为。

### 数据更新频率是多少？

根据系统配置，通常为实时或每日汇总。
