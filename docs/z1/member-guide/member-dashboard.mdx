# 会员数据驾驶舱

## 业务场景

会员数据驾驶舱提供会员业务的可视化数据分析，帮助管理者全面掌握会员运营状况，支持决策分析。

## 前置条件

- 系统中有会员数据
- 具有相应的数据查看权限

## 数据看板

### 核心指标

| 指标 | 说明 |
|------|------|
| 会员总数 | 当前有效会员数量 |
| 本月新增 | 本月新增会员数 |
| 本月活跃 | 本月有操作的会员数 |
| 会员转化率 | 触发会员占比 |

### 会员分布

| 维度 | 说明 |
|------|------|
| 等级分布 | 各等级会员占比 |
| 门店分布 | 各门店会员数量 |
| 导购分布 | 各导购负责客户数 |

### 消费分析

| 指标 | 说明 |
|------|------|
| 累计消费金额 | 会员总消费额 |
| 客单价 | 平均每笔订单金额 |
| 复购率 | 重复购买会员占比 |

## 操作步骤

### 进入数据驾驶舱

1. 进入 **会员管理** → **会员数据驾驶舱**
2. 查看核心数据概览

### 自定义时间筛选

1. 选择时间范围
2. 数据看板自动更新

### 导出报表

点击导出按钮可导出数据分析报表。
