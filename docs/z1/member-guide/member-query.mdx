# 会员查询

## 业务场景

会员查询功能用于根据各种条件筛选和查找会员信息，帮助门店精准定位目标客户，进行会员管理和营销分析。

## 前置条件

- 系统中有会员数据
- 具有相应的查询权限

## 名词解释

| 术语 | 说明 |
|------|------|
| 会员编号 | 会员入会时生成的唯一标识，随机九位数字 |
| 注册日期 | 会员进店后在导购处第一次输入手机号下单的时间 |
| 入会时间 | 会员第一次进入小程序注册的时间 |
| 专属导购 | 1v1进行专业销售服务的专属营业员 |
| 会员标签 | 根据会员喜好制定的品牌偏向标签 |
| 会员积分 | 会员完成特定行为（消费、签到、评价等）获得的虚拟奖励 |
| 会员状态 | 正常、未验证、禁用三个状态 |

### 手机号规则

- 用户手机号受到唯一约束，任意两个用户的手机号不可能重复
- 用户标识符不能为空，但用户手机号可能为空
- 用户注销时，手机号即被清空

### 会员等级

会员通过消费、签到等操作增加经验值进行升级，等级分级为0-10级。

## 操作步骤

### 进入会员查询页面

1. 搜索"会员查询"
2. 或进入 **会员管理** → **会员查询**

### 筛选会员

| 筛选项 | 说明 |
|--------|------|
| 注册日期 | 时间段筛选（包含） |
| 会员姓名 | 模糊匹配 |
| 性别 | 单选：保密/男/女 |
| 联系电话 | 模糊匹配（3位及以上） |
| 最后购买日期 | 时间段筛选（包含） |
| 等级 | 数值范围（0-10） |
| 会员状态 | 单选：正常/未验证/禁用 |
| 会员生日 | 月日时间段筛选 |
| 专属导购 | 多选职员或"无专属导购" |
| 会员标签 | 模糊匹配 |
| 微信绑定 | 选择是否绑定微信 |
| 专属导购绑定 | 已绑定/未绑定 |

### 列表字段

| 字段 | 说明 |
|------|------|
| 入会时间 | 会员注册时间 |
| 会员编号 | 九位唯一标识 |
| 姓名 | 会员名称 |
| 联系电话 | 手机号 |
| 性别 | 男/女/保密 |
| 生日 | 出生日期 |
| 所属部门 | 会员所属门店 |
| 专属导购 | 负责的导购 |
| 等级 | 0-10级 |
| 积分 | 当前积分数量 |
| 状态 | 正常/未验证/禁用 |

### 查看会员详情

点击会员行可查看详细信息：
- 基本信息（姓名、性别、生日、手机号等）
- 运营商信息（运营商、套餐、是否绑定宽带等）
- 购买记录
- 会员积分
- 专属导购

### 导出数据

点击 **导出Excel** 可将查询结果导出。

## 常见问题

### 会员手机号可以重复吗？

不可以。用户手机号受到唯一约束，任意两个用户手机号不能重复。

### 会员编号可以修改吗？

会员编号是系统自动生成的唯一标识，不可修改。

### 手机号可以为空吗？

可以。用户注销时手机号会被清空。
