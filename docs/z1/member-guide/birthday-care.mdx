# 会员生日关怀

## 业务场景

为了能够及时提醒到会员其生日权益，促使会员再次消费，设置会员生日关怀模块。系统支持小程序、短信、企业微信等多种通知方式，并可配置生日礼品券，提升会员体验和忠诚度。

## 前置条件

1. 会员已在小程序中填写并保存过生日信息
2. 如需小程序通知，需获得用户同意通知消息
3. 如需企业微信通知，需配置专属导购关系

## 名词解释

| 术语 | 说明 |
|------|------|
| 会员生日 | 顾客在小程序上注册时或在店内完成订单店员完成会员注册时设置的出生日期 |
| 发送时间 | 设置在会员生日前多少天发送祝福和礼品（0-20天） |
| 祝福模板 | 管理者可自定义生日提醒话术，对不同的提醒方式可以设置不同的祝福模板 |
| 祝福方式 | 可通过小程序、短信、企业微信等方式进行关怀 |
| 提醒人群 | 设置条件筛选会员进行提醒，包括：会员状态、性别、注册日期、最后购买日期、所属部门、等级、绑定专属导购情况 |

## 祝福方式

| 方式 | 说明 |
|------|------|
| 小程序通知 | 需要用户同意通知消息，在会员填写/保存生日时弹窗授权 |
| 短信通知 | 通过短信方式发送生日祝福 |
| 企业微信通知 | 向店长/导购发送通知 |
| 公众号通知 | **目前不做** |

### 小程序通知授权节点

- 会员下订单时：已有授权
- 新注册会员、未填写过生日的老会员填写"生日"字段，点击"保存"后提醒
- 老会员未下过单、填写过生日：不做小程序提醒

## 操作步骤

### 一、开启会员生日关怀

1. 进入 **会员管理** → **会员生日关怀**
2. 打开「提醒开关」

### 二、配置生日提醒设置

1. **设置发送时间**
   - 选择会员生日前多少天发送（0-20天）
   - 含0表示生日当天发送

2. **选择关怀方式**（多选，必填）
   - 小程序通知
   - 短信通知
   - 企业微信通知

3. **配置祝福内容**（根据所选方式填写）

| 关怀方式 | 内容要求 |
|----------|----------|
| 小程序 | 生日祝福（20字以内）+ 温馨提示（20字以内） |
| 短信 | 祝福内容（70字以内） |
| 企业微信 | 祝福内容（50字以内） |

### 三、配置会员生日礼品

1. 选择适用等级
2. 设置礼品卡券（模糊搜索，多选）
   - 卡券失效时会标红显示，提示"xxx卡券已失效，请重新设置"

### 四、设置提醒人群

| 筛选条件 | 选项 |
|----------|------|
| 会员状态 | 正常 / 未验证 / 禁用 |
| 注册日期 | 近一个月 / 三个月 / 半年 / 一年 / 不限 |
| 最后购买 | 近一个月 / 三个月 / 半年 / 一年 / 不限 |
| 性别 | 男 / 女 / 保密 |
| 所属部门 | 多选 |
| 等级范围 | 最小值～最大值（仅影响会员关怀通知，不影响卡券发放） |
| 专属导购绑定 | 已绑定 / 未绑定 |

### 五、企业微信通知配置

#### 店长通知

- 每月1日执行定时任务
- 查询当月过生日的会员
- 向门店店长发送企微/PC通知
- 发送时间：每月1日上午09:00
- 通知标题：会员生日通知
- 通知内容：您有会员今天过生日，快去送上祝福吧！点击"去处理"跳转"会员生日关怀列表"页面

#### 导购通知

- 会员生日当天上午09:00发送
- 通过企业微信向专属导购发送「客户联系」通知
- 导购需手动确认后发送生日祝福

#### 通知规则

1. 查找会员最后一条订单记录（已出库/已完成/已评价）
2. 根据订单所在门店查找最后添加的导购作为专属导购
3. 无订单记录则找最后添加的导购
4. 导购需为在职状态

#### 通知发送优先级

| 场景 | 处理方式 |
|------|----------|
| 有消费、有导购企微 | 从专属导购企微发送 |
| 有消费、无导购企微 | 不发送企微通知 |
| 无消费、有导购企微 | 从最后添加的导购企微发送 |
| 无消费、无导购企微 | 不发送企微通知 |

### 六、专属导购更新规则

#### 添加专属导购时

根据会员是否消费判断是否更新专属导购。

#### 删除专属导购时

根据会员消费记录和最近添加的专属导购进行更新。

## 常见问题

### 1. 为什么会员没有收到生日通知？

请检查以下条件：
- 会员是否在小程序中填写并保存了生日信息
- 会员状态是否正常
- 会员是否符合提醒人群设置的条件
- 小程序通知需要用户同意授权

### 2. 企业微信通知发送给谁？

系统根据会员的订单记录和导购关系确定专属导购，然后从该导购的企业微信发送通知。

### 3. 多个门店消费的会员通知发给谁？

根据以下优先级确定：
- 最后消费门店的最后添加导购企微发送
- 若无消费，仅添加过导购企微，则从最后添加的导购企微发送

### 4. 卡券失效了怎么办？

卡券失效时会标红显示，并提示"xxx卡券已失效，请重新设置"。
