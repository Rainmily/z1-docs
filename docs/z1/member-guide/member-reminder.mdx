# 会员定时提醒

## 业务场景

会员定时提醒功能用于针对用户创建定时提醒预案，系统自动执行提醒操作：

- **活动提醒**：新店开业、节日促销等活动通知
- **优惠提醒**：会员专享优惠、限时折扣等信息推送
- **新机上市**：新品到货、热门机型推荐等通知

支持短信、公众号、小程序等多种提醒方式。

## 前置条件

- 已配置短信/公众号/小程序提醒模板
- 需要有可发送的会员人群

## 名词解释

| 术语 | 说明 |
|------|------|
| 会员定时提醒 | 程序根据设定的提醒预案进行自动提醒操作 |
| 提醒预案 | 人为创建的提醒预案，设定需要进行的提醒操作 |
| 提醒类型 | 活动提醒 / 优惠提醒 / 新机上市 |
| 提醒方式 | 短信 / 公众号 / 小程序 |
| 提醒预案号 | 每个提醒预案的单号 |

### 提醒预案号生成规则

```
日期（年月日时分）+ 序号（六位，如000001）
```

## 小程序模板

**新活动通知**
- 模板编号：11550
- 模板分类：办公
- 模板字段：
  - 活动名称
  - 活动概述
  - 活动开始时间
  - 活动结束时间
  - 温馨提示

## 提醒状态

| 状态 | 说明 |
|------|------|
| 待提醒 | 预计建立了提醒预案，但是还没进行提醒 |
| 提醒成功 | 系统完成提醒操作，提醒预案状态变更为"提醒成功" |
| 提醒终止 | 在未开始提醒前取消了提醒预案 |

## 提醒流程

```
待提醒 → 提醒成功
    ↓
提醒终止
```

## 操作步骤

### 创建提醒预案

1. 进入 **会员管理** → **会员定时提醒**
2. 点击 **新建提醒预案**

#### 填写基本信息

| 字段 | 说明 |
|------|------|
| 提醒类型 | 活动提醒 / 优惠提醒 / 新机上市 |
| 提醒方式 | 短信 / 公众号 / 小程序（可多选） |
| 提醒时间 | 设置发送时间（Unix时间戳） |
| 提醒内容 | 根据选择的模板填写内容 |
| 备注 | 此次提醒预案的生成原因 |

#### 选择提醒人群

选择需要接收提醒的会员范围（UserIdent数组）。

### 查看提醒预案列表

支持筛选条件：
- 提醒类型（多选）
- 提醒预案号（模糊匹配）
- 创建人（多选）
- 状态（多选）
- 创建时间范围
- 提醒时间范围

### 查看提醒详情

查看单条提醒预案的详细信息。

### 修改提醒预案

- 提醒执行前1小时可修改
- 状态必须为"待提醒"

### 取消提醒

- 提醒执行前1小时可取消
- 需要填写取消备注

### 提醒成功

系统验证状态为"待提醒"后，执行提醒操作，状态变更为"提醒成功"。

## 常见问题

### 提醒可以撤回吗？

不可以。提醒成功后将无法撤回。

### 修改有时间限制吗？

提醒执行前1小时可以修改提醒预案。

### 取消提醒需要什么条件？

状态必须为"待提醒"，且需要填写取消备注。
