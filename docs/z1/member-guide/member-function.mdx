# 会员功能

## 业务场景

会员功能用于管理门店会员体系，实现以下核心目标：

- **会员流量池运营**：打造会员流量池，维护用户关系，提升品牌影响
- **复购管理**：统计会员复购订单与复购率，支持复购管理功能
- **精准营销**：识别头部顾客，进行各门店复购率统计

## 前置条件

- 已开通会员模块权限
- 系统中存在会员数据
- 如需查看专属导购相关功能，需先配置职员信息

## 名词解释

### 会员类型

| 术语 | 说明 |
|------|------|
| 会员 | 所有在系统中注册的用户均可以作为会员 |
| 已触发会员 | 在门店产生过消费的会员（会员等级：青铜→白银→黄金→铂金→钻石→星钻，又称SVIP） |
| 未触发会员 | 未在门店产生过消费的会员（会员等级：普通会员，又称VIP） |

### 会员状态

| 状态 | 说明 |
|------|------|
| 正常状态 | 1年内在门店产生≥1笔消费的已触发会员 |
| 沉默状态 | 曾经产生过消费，但消费时间超过1年的已触发会员 |
| 复购状态 | 1年内产生≥2笔消费的已触发会员 |

### 会员等级体系

会员等级按消费累计金额划分为六个等级：

| 等级 | 名称 |
|------|------|
| 青铜 | SVIP |
| 白银 | SVIP |
| 黄金 | SVIP |
| 铂金 | SVIP |
| 钻石 | SVIP |
| 星钻 | SVIP |

### 会员归属

| 术语 | 说明 |
|------|------|
| 有专属导购会员 | 拥有专属导购的已触发会员 |
| 无专属导购会员 | 没有专属导购的已触发会员（考虑职员离职、调岗等情况） |
| 专属导购 | 1v1进行专业销售服务的专属营业员 |

## 操作步骤

### 查看已触发会员列表

1. 进入 **会员管理** → **已触发会员列表**
2. 默认展示正常、沉默、复购状态的会员数据

#### 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 是否有专属导购 | 是 / 否 |
| 专属导购 | 选择导购姓名，或选择"无专属导购" |
| 会员状态 | 正常 / 沉默 / 复购（单选） |
| 最后购买时间 | 时间段筛选 |
| 累计消费金额 | 金额区间（最小～最大） |
| 最后消费金额 | 金额区间（最小～最大） |
| 手机号 | 模糊匹配（需输入3位及以上） |
| 会员等级 | 单选 |

#### 列表字段说明

| 字段 | 说明 |
|------|------|
| 序号 | 按自然顺序排列 |
| 用户名 | 会员名称 |
| 手机号 | 联系方式 |
| 累计消费金额 | 该会员累计消费总额（含退换货） |
| 最后购买金额 | 最近一笔订单金额 |
| 最后购买日期 | 格式：YYYY-MM-DD HH:mm:ss |
| 专属导购 | 负责该会员的导购人员 |
| 是否有专属导购 | 是 / 否 |
| 状态 | 正常 / 沉默 / 复购 |
| 会员等级 | 青铜/白银/黄金/铂金/钻石/星钻 |
| 操作 | 查看 / 编辑 |

> **注意**：累计消费金额包含退换货情况。当金额为0时，可能是产生了"一买一退"的交易。

### 查看未触发会员列表

1. 进入 **会员管理** → **未触发会员列表**

#### 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 手机号 | 模糊匹配 |
| 注册门店 | 下拉选择 |
| 注册店员 | 选择职员或"无专属导购" |
| 真实姓名 | 模糊匹配 |
| 性别 | 下拉选择 |

#### 列表字段说明

| 字段 | 说明 |
|------|------|
| 序号 | 按自然顺序排列 |
| 用户名 | 会员名称 |
| 真实姓名 | 真实姓名 |
| 注册日期 | 入会时间 |
| 注册门店 | 初次注册时的专属导购门店 |
| 手机号 | 联系方式 |
| 会员等级 | 普通会员 |
| 最后登录日期 | 最近一次登录时间 |

| 会员性别 | 下拉选择 |

| 会员邮箱 | 输入框 |
| 运营商 | 下拉选择 |
| 运营商套餐 | 输入框 |
| 是否绑定宽带 | 下拉选择 |
| 重流量轻通话 | 下拉选择 |

#### 不可编辑字段（仅查看）

### 查看会员购买记录

1. 在会员详情页点击 **会员订单** tab（默认选中购买记录）

#### 购买记录列表字段

| 字段 | 说明 |
|------|------|
| 订单日期 | 格式：YYYY-MM-DD HH:mm:ss |
| 销售类型 | 订单类型为网销单时点击打开商城订单详情 |
| 销售部门 | 所属门店 |
| 订单金额 | 网销单显示商城订单金额，其他显示实际金额 |

#### 商品信息展示

| 字段 | 说明 |
|------|------|
| 序号 | 商品顺序号 |
| 商品名称 | 商品名称 |
| 商品编号 | SKU编码 |
| 单价 | 非标货品单价为库存价 |
| 折扣后价格 | 非标货品：库存价 - 代金券作用在非标上的价格 |
| 数量 | 购买数量 |

- 若存在服务，显示服务子表（服务名称、单价、折扣后价格）
- 若存在赠品，显示赠品子表（赠品名称、类型、单价、数量）

## 常见问题

### 累计消费金额为0是什么原因？

累计消费金额包含退换货情况。当金额为0时，可能是会员第一次消费时产生了"一买一退"的交易。

### 会员状态自动变更规则

系统每天在维护期自动更新会员状态：

- **→沉默**：最近购买时间距今超过1年
- **→复购**：最近二次购买时间距今小于1年
- **→正常**：最近二次购买时间距今超过1年或无二次购买，且最近购买时间距今小于1年

### 未触发会员如何转化为已触发会员？

未触发会员在门店产生第一笔消费订单后，将自动转为已触发会员，等级为青铜。

### 会员手机号可以修改吗？

会员手机号不可直接修改。如需更换手机号，需通过"解绑手机号"功能处理。

## 状态变更示例

**场景1：正常→沉默**

会员于2023年9月7日产生消费，且在2024年9月7日前没有第二次消费：
- 2024年9月7日及之前：状态为正常
- 2024年9月8日起：状态变更为沉默

**场景2：正常→复购→沉默**

会员于2023年9月7日消费，2023年12月7日再次消费，此后无第三次消费：
- 2023年12月7日：状态变更为复购
- 2024年9月8日：状态变更为正常
- 2024年12月8日：状态变更为沉默
