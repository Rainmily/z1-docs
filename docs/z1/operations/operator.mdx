# 运营商模块

## 业务场景

运营商模块用于管理与运营商相关的业务，如号码入网、套餐办理等。

## 前置条件

- 运营商账户已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 运营商 | 移动/联通/电信 |
| 套餐 | 资费套餐 |
| 号码 | 手机号码 |

## 操作步骤

### 号码入网

1. 选择运营商
2. 选择号码
3. 办理套餐
4. 确认入网

### 套餐变更

1. 选择要变更的号码
2. 选择新套餐
3. 确认变更
