# 职员模块

## 业务场景

职员模块是人员管理的基础，用于管理企业职员信息，包括入职、离职、岗位调整等全生命周期管理。

## 前置条件

- 部门信息已配置
- 职务职级已设置

## 名词解释

| 术语 | 说明 |
|------|------|
| 职员 | 企业正式员工 |
| 在职 | 当前在岗状态 |
| 离职 | 已离开企业 |
| 职级 | 职员的专业等级 |

## 操作步骤

### 新增职员

1. 进入 **人事管理** → **职员管理**
2. 点击 **新增职员**
3. 填写基本信息（姓名、工号、部门等）
4. 设置职务职级
5. 关联用户账号
6. 保存

### 修改职员信息

1. 找到要修改的职员
2. 点击编辑
3. 修改相关信息
4. 保存

### 职员列表

| 功能 | 说明 |
|------|------|
| 查看列表 | 按条件筛选查看职员 |
| 在职一览表 | 查看当前在职职员 |

### 职员状态

| 状态 | 说明 |
|------|------|
| 在职 | 正常工作状态 |
| 离职 | 已离开企业 |

## 常见问题

### 职员必须属于部门吗？

是的，职员必须属于一个部门。

### 职员与用户是什么关系？

每个职员都关联一个用户账号，用于系统登录。
