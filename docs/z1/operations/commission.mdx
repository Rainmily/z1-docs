# 提成基础参数

## 业务场景

提成基础参数用于设置销售提成计算规则，包括提成比例、阶梯提成等。

## 前置条件

- 商品分类已配置
- 职员信息已录入

## 提成计算方式

| 方式 | 说明 |
|------|------|
| 固定比例 | 按销售额固定比例计算 |
| 阶梯提成 | 销售额越高提成比例越高 |
| 单品提成 | 按商品设置单独提成 |

## 操作步骤

### 配置提成规则

1. 进入 **系统设置** → **提成参数**
2. 选择提成计算方式
3. 设置提成比例
4. 保存
