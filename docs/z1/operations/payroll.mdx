# 工资表

## 业务场景

工资表结合绩效、提成功能以及考勤数据，计算员工精确工资，包括基本工资、工龄工资、绩效工资、补贴等组成部分。

## 前置条件

- 绩效模块已配置
- 提成数据已生成
- 考勤数据已同步

## 工资构成

| 项目 | 说明 |
|------|------|
| 基本工资 | 职员固定月度工资（必填） |
| 工龄工资 | 根据入职/转正时间计算 |
| 绩效浮动工资 | 根据KPI考核结果计算 |
| 补贴合计 | 固定补贴 + 福利补贴 |
| 提成 | 销售业绩提成 |

## 工龄工资计算

### 规则

- 从入职日期或转正日期开始计算
- 每在职 n 个月，工龄工资增加 m 元
- 当月不算工龄工资，次月1日开始计算

### 计算公式

```
工龄工资 = floor((工资月份 - 入职月份) / n) * m
```

### 示例

```
入职时间：2023年5月10日
n=1，m=1

2023年6月：工龄工资 = 0
2023年7月：工龄工资 = 1
```

## 绩效浮动工资

### 计算公式

```
绩效浮动工资 = 基本工资 × KPI关联比例 × (个人绩效系数 - 1)
```

### 示例

```
基本工资：4000
KPI关联比例：0.5
绩效系数：1.2

关联绩效部分 = 4000 × 0.5 = 2000
绩效浮动工资 = 2000 × 1.2 - 2000 = 400
```

## 操作步骤

### 查看工资表

1. 进入 **人事管理** → **工资表**
2. 选择工资月份
3. 选择部门/职员
4. 查看工资明细

### 导出工资报表

1. 查看工资表后
2. 点击导出
3. 选择格式（Excel/PDF）
4. 保存报表

## 常见问题

### 工龄从什么时候开始算？

可选择从入职日期或转正日期开始计算。

### 绩效为空怎么办？

绩效为空时，绩效浮动工资取0。
