# 门店管理

## 业务场景

门店管理用于管理企业门店信息，包括基本信息、形象展示、展位布局等。每个门店对应一个部门，但不是所有部门都是门店。

## 前置条件

- 部门信息已创建

## 名词解释

| 术语 | 说明 |
|------|------|
| 基本信息 | 门店名称、编号、类型、负责人 |
| 扩展信息 | 非必填的其他信息 |
| 门店主图 | 门店门脸正面照片（1:1比例） |
| 地理坐标 | 门店位置经纬度 |

## 门店信息

### 基本信息（必填）

| 字段 | 说明 |
|------|------|
| 门店名称 | 默认取部门名称 |
| 门店地址 | 默认取部门地址 |
| 门店电话 | 默认取部门电话 |
| 门店主图 | 门脸正面照（1:1） |
| 营业时间 | 20字以内 |

### 扩展信息（非必填）

| 字段 | 说明 |
|------|------|
| 责任城管 | 片区负责人 |
| 责任城管电话 | 联系电话 |
| 门店备注 | 其他备注信息 |
| 周边房价 | 周边房价线索 |

### 形象展示

| 类型 | 说明 |
|------|------|
| 周边引导照 | 周边标志性建筑引导（多张） |
| 门店形象照 | 室外、室内现场照（多张） |
| 门店设计图 | 原始户型设计图（多张） |
| 展台布局图 | 展位及体验桌柜布局（多张） |

## 操作步骤

### 新增门店

1. 进入 **人事管理** → **门店管理**
2. 点击 **新增门店**
3. 填写基本信息
4. 上传门店图片
5. 保存

### 编辑门店

1. 找到要编辑的门店
2. 点击编辑
3. 修改相关信息
4. 保存

### 门店一览表

查看所有门店的列表信息。

## 常见问题

### 门店和部门是什么关系？

门店必须是部门，部门不一定是门店。

### 门店主图有什么要求？

需要上传前裁剪，长宽比1:1。
