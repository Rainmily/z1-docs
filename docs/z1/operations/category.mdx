# 分类模块

## 业务场景

分类模块用于管理商品和内容的分类体系，建立清晰的层级结构，方便数据管理和检索。适用于商品分类、内容分类等多种场景。

## 前置条件

无

## 名词解释

| 术语 | 说明 |
|------|------|
| 分类 | 具有层级结构的分类体系 |
| 上级分类 | 上一级分类节点 |
| 末级分类 | 没有下级的分类 |

## 分类类型

| 类型 | 说明 |
|------|------|
| 商品分类 | 用于商品管理 |
| 内容分类 | 用于文章/内容管理 |
| 品牌分类 | 用于品牌管理 |

## 操作步骤

### 创建分类

1. 进入 **系统设置** → **分类模块**
2. 选择分类类型
3. 点击 **新增分类**
4. 填写分类名称
5. 选择上级分类（可选）
6. 设置排序
7. 保存

### 编辑分类

1. 找到要编辑的分类
2. 点击编辑
3. 修改分类信息
4. 保存

### 管理分类

| 操作 | 说明 |
|------|------|
| 调整顺序 | 修改分类排列顺序 |
| 停用分类 | 停用后不可使用 |
| 启用分类 | 恢复使用停用的分类 |
| 删除分类 | 删除空分类（无下级） |

### 分类迁移

将下级分类迁移到其他上级分类下。

## 常见问题

### 删除分类有影响吗？

删除分类前需确保没有下级分类和关联商品。

### 分类可以调整顺序吗？

可以，拖拽或设置排序号调整顺序。

### 停用的分类还能显示吗？

停用后该分类及其下级不再显示。
