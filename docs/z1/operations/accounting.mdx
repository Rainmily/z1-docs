# 会计科目管理

## 业务场景

会计科目管理用于设置和管理企业会计科目体系，规范财务核算。

## 前置条件

无

## 名词解释

| 术语 | 说明 |
|------|------|
| 资产类科目 | 资产类会计科目 |
| 负债类科目 | 负债类会计科目 |
| 损益类科目 | 收入支出科目 |

## 操作步骤

### 查看科目列表

1. 进入 **系统设置** → **会计科目**
2. 查看科目体系

### 新增科目

1. 选择上级科目
2. 填写科目信息
3. 保存
