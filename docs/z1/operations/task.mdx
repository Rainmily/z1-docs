# 任务管理

## 业务场景

任务管理用于分配和跟踪工作任务，提高团队协作效率。

## 前置条件

无

## 名词解释

| 术语 | 说明 |
|------|------|
| 任务 | 具体的工作事项 |
| 负责人 | 任务执行人 |
| 截止日期 | 任务完成期限 |

## 操作步骤

### 创建任务

1. 进入 **企业管理** → **任务管理**
2. 点击 **新建任务**
3. 填写任务内容
4. 指定负责人
5. 设置截止日期
6. 保存

### 完成任务

1. 点击任务
2. 更新进度
3. 标记完成
