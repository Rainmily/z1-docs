# 绩效模块

## 业务场景

绩效模块用于管理员工绩效考核，根据KPI和业绩系数计算绩效工资。

## 前置条件

- 绩效考核指标已配置
- 职员信息已录入

## 绩效构成

| 项目 | 说明 |
|------|------|
| 绩效系数 | 个人工作表现评分 |
| 业绩系数 | 完成业绩情况评分 |
| 最终系数 | 绩效系数 × 业绩系数 |

## 操作步骤

### 设置考核指标

1. 进入 **人事管理** → **绩效模块**
2. 配置考核指标
3. 设置权重

### 绩效评定

1. 期初设置绩效目标
2. 期末进行绩效评定
3. 计算绩效工资
