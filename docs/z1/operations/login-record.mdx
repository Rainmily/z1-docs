# 职员登录记录

## 业务场景

职员登录记录用于审计职员的登录行为，保障系统安全。

## 前置条件

无

## 记录内容

| 字段 | 说明 |
|------|------|
| 登录时间 | 登录时间 |
| 登录方式 | 账号密码/扫码 |
| IP地址 | 登录IP |
| 设备信息 | 登录设备 |

## 操作步骤

### 查看登录记录

1. 进入 **系统设置** → **登录记录**
2. 选择时间范围
3. 选择职员
4. 查看记录
