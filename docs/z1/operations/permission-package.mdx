# 权限包

## 业务场景

权限包用于管理职员的功能权限，通过权限包批量分配权限。

## 前置条件

- 职员信息已录入

## 名词解释

| 术语 | 说明 |
|------|------|
| 权限包 | 权限的集合 |
| 功能权限 | 具体的功能操作权限 |

## 操作步骤

### 创建权限包

1. 进入 **系统设置** → **权限管理**
2. 点击 **新建权限包**
3. 添加权限项
4. 保存

### 分配权限包

1. 选择职员
2. 选择权限包
3. 确认分配
