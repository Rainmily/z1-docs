# 工分管理

## 业务场景

工分管理用于记录员工的工作积分，作为绩效参考。

## 前置条件

- 工分规则已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 工分 | 工作量积分 |
| 工分规则 | 积分获取规则 |

## 操作步骤

### 记录工分

1. 进入 **企业管理** → **工分管理**
2. 点击 **记录工分**
3. 选择职员
4. 填写工分和原因
5. 保存

### 查看工分

1. 查看职员工分明细
2. 统计周期汇总
3. 导出报表
