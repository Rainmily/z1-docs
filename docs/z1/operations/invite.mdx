# 邀请模块

## 业务场景

邀请模块用于管理营销活动邀请，记录邀请人和被邀请人关系。

## 前置条件

- 邀请码已配置

## 名词解释

| 术语 | 说明 |
|------|------|
| 邀请码 | 区分邀请渠道的标识 |
| 邀请人 | 发起邀请的人 |
| 被邀请人 | 接收邀请的人 |

## 操作步骤

### 创建邀请活动

1. 进入 **营销管理** → **邀请模块**
2. 点击 **新建活动**
3. 设置邀请规则
4. 生成邀请码

### 查看邀请记录

1. 查看邀请数据
2. 统计邀请效果
3. 分析转化率
