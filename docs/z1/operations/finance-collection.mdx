# 财务代收

## 业务场景

财务代收用于管理代收款项，如代收货款、代收费用、押金等。系统记录代收明细，便于财务对账和凭证生成。

## 前置条件

- 账户信息已配置
- 代收权限已开通

## 代收类型

| 类型 | 说明 |
|------|------|
| 代收货款 | 代替供应商收取的货款 |
| 代收费用 | 代收各类服务费用 |
| 押金 | 收取的押金款项 |
| 其他 | 其他代收款项 |

## 操作步骤

### 创建代收单

1. 进入 **财务管理** → **财务代收**
2. 点击 **新建代收单**
3. 选择代收类型
4. 填写付款方信息
5. 填写代收金额
6. 添加备注说明
7. 提交审核

### 代收核销

1. 收到代收款项后
2. 选择要核销的代收单
3. 确认核销金额
4. 选择收款账户
5. 完成核销

### 生成凭证

核销后系统自动生成收款凭证，关联到财务账务。

## 代收状态

| 状态 | 说明 |
|------|------|
| 待核销 | 已创建，等待收款 |
| 部分核销 | 已部分收款 |
| 已核销 | 款项已收齐 |
| 已取消 | 代收已取消 |

## 常见问题

### 代收可以部分核销吗？

可以，支持部分核销。

### 核销后可以撤销吗？

核销后如需撤销，需冲销处理。
