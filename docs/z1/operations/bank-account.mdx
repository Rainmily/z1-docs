# 银行账户

## 业务场景

银行账户用于管理企业银行账户信息，进行资金收付管理。

## 前置条件

- 银行账户已开通

## 名词解释

| 术语 | 说明 |
|------|------|
| 账户类型 | 对公/对私 |
| 账户余额 | 当前可用余额 |

## 操作步骤

### 新增账户

1. 进入 **系统设置** → **银行账户**
2. 点击 **新增账户**
3. 填写账户信息
4. 保存

### 账户管理

1. 查看账户余额
2. 管理账户状态
3. 设置默认账户
