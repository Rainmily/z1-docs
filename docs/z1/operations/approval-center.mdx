# 审批流程中心

## 业务场景

审批流程中心（S1）是自主设计的审批模块，用于集中处理系统内所有需要审批的业务，如采购订单、优惠审批、门店调整等。

## 前置条件

- 审批模板已配置
- 审批节点已设置

## 名词解释

| 术语 | 说明 |
|------|------|
| 审核 | 审查核实，无最终决定权 |
| 审批 | 审查并批准，有决定权 |
| 审批表单 | 需要审批的业务内容 |
| 审批节点 | 审批流程的步骤 |
| 审批模板 | 表单+节点的完整流程 |

## 审批组件

| 组件类型 | 说明 |
|----------|------|
| 文本 | 文本输入 |
| 数字 | 数值输入（含单位、精度） |
| 金额 | 金额输入 |
| 人员 | 人员选择 |
| 时间 | 日期时间选择 |
| 附件 | 文件上传（限5个，单个10MB） |
| 表格 | 多列数据表格 |

## 审批条件

| 条件类型 | 说明 |
|----------|------|
| 人员 | 发起人是否属于某职员/部门 |
| 数字 | 大小比较判断 |
| 金额 | 金额范围判断 |

## 操作步骤

### 创建审批模板

1. 进入 **系统设置** → **审批流程中心**
2. 点击 **新建模板**
3. 设计审批表单
4. 配置审批节点
5. 保存模板

### 提交审批

1. 选择审批模板
2. 填写审批表单
3. 提交审批
4. 等待审批结果

### 处理审批

1. 收到审批通知
2. 查看审批内容
3. 通过/驳回审批

## 常见问题

### 最多能创建多少模板？

限制最大模板数量为200条。

### 系统业务模板可以修改吗？

不可以修改表单内容，但可以编辑审批节点。
