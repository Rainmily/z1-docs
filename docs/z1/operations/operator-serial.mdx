# 运营商序列号管理

## 业务场景

运营商序列号管理用于管理运营商SIM卡的序列号（ICCID）流转。

## 前置条件

- SIM卡已入库

## 名词解释

| 术语 | 说明 |
|------|------|
| ICCID | SIM卡序列号 |
| 号码状态 | 占用/空闲/注销 |

## 操作步骤

### 录入序列号

1. 进入 **运营商管理** → **序列号管理**
2. 扫描/录入ICCID
3. 关联号码
4. 保存

### 序列号状态变更

1. 选择序列号
2. 修改状态
3. 确认变更
