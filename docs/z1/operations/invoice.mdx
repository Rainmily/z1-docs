# 发票管理

## 业务场景

发票管理用于管理企业发票的开具、作废、红冲等操作。

## 前置条件

- 税号已配置
- 发票权限已开通

## 名词解释

| 术语 | 说明 |
|------|------|
| 增值税发票 | 可抵扣进项税的发票 |
| 普通发票 | 不可抵扣的发票 |

## 操作步骤

### 开具发票

1. 进入 **财务管理** → **发票管理**
2. 选择订单
3. 确认发票信息
4. 开具发票

### 发票作废/红冲

1. 选择要作废的发票
2. 确认作废原因
3. 完成作废
