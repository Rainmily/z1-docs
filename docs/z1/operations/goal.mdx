# 目标管理

## 业务场景

目标管理用于设定和跟踪团队/个人目标，对齐企业战略。

## 前置条件

无

## 名词解释

| 术语 | 说明 |
|------|------|
| 目标 | 具体的工作目标 |
| 衡量指标 | KPI/OKR等 |
| 进度 | 完成情况 |

## 操作步骤

### 创建目标

1. 进入 **企业管理** → **目标管理**
2. 点击 **新建目标**
3. 填写目标内容
4. 设置衡量指标
5. 指定负责人
6. 保存

### 跟踪进度

1. 定期更新进度
2. 记录里程碑
3. 完成评估
