# 财务支出单

## 业务场景

财务支出单用于记录企业各类支出，如采购付款、费用报销等。

## 前置条件

- 账户余额充足

## 名词解释

| 术语 | 说明 |
|------|------|
| 支出类型 | 采购/费用/其他 |
| 付款方式 | 现金/转账 |

## 操作步骤

### 创建支出单

1. 进入 **财务管理** → **财务支出单**
2. 点击 **新建支出**
3. 选择支出类型
4. 填写金额和摘要
5. 选择付款账户
6. 提交
