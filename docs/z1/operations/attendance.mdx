# 考勤模块

## 业务场景

考勤模块用于记录和管理职员考勤数据，支持对接第三方考勤系统（企业微信、钉钉、飞书等），展示打卡记录和月统计信息。

## 前置条件

- 已对接第三方考勤系统
- 职员信息已录入

## 名词解释

| 术语 | 说明 |
|------|------|
| 考勤记录 | 每次打卡的明细数据 |
| 月统计 | 每月考勤数据的汇总 |

## 考勤数据

| 字段 | 说明 |
|------|------|
| 打卡时间 | 职员打卡的具体时间 |
| 打卡类型 | 上班/下班/外勤等 |
| 打卡地点 | 位置信息 |
| 考勤状态 | 正常/迟到/早退/缺勤等 |

## 操作步骤

### 查看考勤记录

1. 进入 **人事管理** → **考勤管理**
2. 选择职员或部门
3. 选择日期范围
4. 查看打卡明细

### 查看月统计

1. 选择考勤月份
2. 查看月度考勤汇总
3. 导出考勤报表

## 数据对接

| 第三方系统 | 说明 |
|------------|------|
| 企业微信 | 对接企业微信考勤 |
| 钉钉 | 对接钉钉考勤 |
| 飞书 | 对接飞书考勤 |

## 常见问题

### 考勤设置在哪里配置？

考勤相关设置不在Z1系统中配置，由第三方系统管理。

### 支持哪些考勤类型？

支持上班、下班、外勤等多种打卡类型。
