# 维修文档

## 业务场景

维修文档用于管理商品维修记录，包括维修状态机和维修流程。

## 前置条件

- 维修工单已创建

## 维修状态

| 状态 | 说明 |
|------|------|
| 待接单 | 等待维修接单 |
| 维修中 | 正在进行维修 |
| 已完成 | 维修完成 |
| 已取机 | 客户已取机 |

## 操作步骤

### 创建维修单

1. 进入 **维修管理** → **维修文档**
2. 点击 **新建维修单**
3. 填写故障信息
4. 提交

### 维修流程

1. 接单
2. 诊断
3. 维修
4. 测试
5. 完成
