# 表单模块

## 业务场景

表单模块用于创建自定义表单，收集各类业务数据。

## 前置条件

无

## 表单组件

| 组件 | 说明 |
|------|------|
| 文本框 | 单行/多行文本 |
| 数字框 | 数值输入 |
| 日期选择 | 日期/时间 |
| 下拉选择 | 选项选择 |
| 文件上传 | 图片/附件 |

## 操作步骤

### 创建表单

1. 进入 **系统设置** → **表单模块**
2. 点击 **新建表单**
3. 拖拽添加组件
4. 设置表单属性
5. 保存发布

### 查看表单数据

1. 进入表单详情
2. 查看已提交数据
3. 导出数据
