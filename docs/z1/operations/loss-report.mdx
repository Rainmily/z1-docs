# 报损单

## 业务场景

报损单用于记录商品报损情况，处理因损坏、过期、变质等原因造成的库存损耗，需要经审批后核销库存。

## 前置条件

- 报损权限已配置
- 有可报损的库存商品

## 名词解释

| 术语 | 说明 |
|------|------|
| 报损 | 商品因非正常原因造成的库存核销 |
| 报损原因 | 导致报损的具体原因 |
| 报损金额 | 报损商品的价值 |

## 报损原因

| 原因类型 | 说明 |
|----------|------|
| 损坏 | 商品物理损坏 |
| 过期 | 商品超过保质期 |
| 变质 | 商品品质发生变化 |
| 丢失 | 库存盘点缺失 |
| 其他 | 其他合理原因 |

## 操作步骤

### 创建报损单

1. 进入 **库存管理** → **报损单**
2. 点击 **新建报损单**
3. 选择报损仓库
4. 添加报损商品
5. 填写报损数量
6. 选择报损原因
7. 填写备注说明
8. 提交审核

### 审核报损

1. 审核人员查看报损单
2. 核对商品和数量
3. 审批通过/驳回
4. 通过后库存相应减少

### 生成凭证

审核通过后，系统自动生成报损凭证，用于财务核算。

## 常见问题

### 报损需要哪些权限？

需要有报损权限才能创建报损单。

### 报损金额如何计算？

报损金额 = 报损数量 × 成本单价。

### 报损后可以恢复吗？

不可以，报损是不可逆操作。
