# Z1使用说明书

## 业务场景

Z1使用说明书是Z1系统的入门指南，帮助用户快速了解系统功能。

## 内容概要

### 系统概述

Z1是掌上乾坤推出的手机连锁门店管理系统，提供进销存、会员营销、财务管理等一体化解决方案。

### 主要模块

| 模块 | 说明 |
|------|------|
| 销售管理 | 零售、批发、订单管理 |
| 会员营销 | 会员、积分、营销活动 |
| 库存管理 | 采购、调拨、盘点 |
| 财务管理 | 收款、付款、账务 |
| 人事管理 | 职员、考勤、绩效 |

### 快速入门

1. 登录系统
2. 配置基础数据（部门、商品、职员）
3. 开始使用各模块功能

### 帮助支持

- 遇到问题可查看各模块操作手册
- 联系技术支持获取帮助
