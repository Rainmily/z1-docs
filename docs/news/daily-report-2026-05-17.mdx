# 华为动态速递（5月17日）

## 市场概述

5月17日，手机圈迎来重磅信号：卢伟冰预告下半年国产直板旗舰价格将突破1万元，小米Air工程机被砍原因曝光，运营商回应"免月租"传言，华为问界M9预订突破5万台，三星推出全新混合键轴笔记本。另外有爆料称4月iPhone在北美超越三星重夺榜首。

## 小米密集爆料

### 卢伟冰：国产直板旗舰价格将破万

卢伟冰今日透露：今年下半年可能一些国产旗舰直板手机价格突破1万元。他同时确认，玄戒芯片今年肯定会迭代，一定是一块非常强的芯片；miclaw不会取代小爱同学，未来会合体赋能。![](https://www.ithome.com/0/951/380.htm)

**点评**：万元以上的国产直板旗舰，这个信号非常重要。如果顶配版本真的站上这个价位段，说明国产旗舰在影像、芯片、AI等维度已经和苹果Pro Max站在同一价位线竞争了。

### 小米Air工程机被砍原因

卢伟冰回应为什么没有出iPhone Air形态的产品：上市前被砍，续航和性能影响体验。同期曝光的小米Air超轻薄旗舰工程机规格显示，该机型重量仅980g，搭载定制8寸OLED屏，最终因续航问题被砍。![](https://www.ithome.com/0/951/399.htm)

## 运营商回应"免月租"传言

### 联通推按量计费

三大运营商回应"即将免月租"传言：联通推出按量计费模式，移动和电信暂无计划。这一表态意味着基础语音和流量月租模式短期内不会取消。![](https://www.ithome.com/0/951/378.htm)

**点评**：运营商在存量竞争下压力越来越大，按量计费是个合理的差异化方向。但对用户而言，有月租的心理安全感是按量计费无法替代的。

## 问界M9预订超5万台

### 预售49.98万起

鸿蒙智行全新一代问界M9系列预订量已突破5万台，预售49.98万元起，搭载全新华为途灵龙行平台。![](https://www.ithome.com/0/951/382.htm)

## 三星新产品线

### 混合键轴笔记本

三星推出全新混合键轴笔记本，兼顾机械键盘手感和静音需求。![](https://www.ithome.com/0/951/396.htm)

## OPPO Find X9 Ultra屏幕问题

### 白色变粉色 OTA修复

OPPO Find X9 Ultra被用户发现在拍视频时白色偏粉色，客服回应称本月下旬通过OTA修复。![](https://www.ithome.com/0/951/248.htm)

## 今日总结

1. **小米价格信号**：卢伟冰确认玄戒芯片今年迭代，同时预告国产旗舰直板手机价格将破万元
2. **问界M9热销**：预订量突破5万台，49.98万起的预售成绩超出市场预期
3. **运营商转型**：联通推按量计费，移动电信暂无免月租计划，传统月租模式仍为主流
4. **OPPO屏幕门**：Find X9 Ultra白色偏粉色问题本月下旬OTA修复
