# 手机行业日报（5月27日）

## 市场概述

今日手机市场迎来两件大事：vivo Y600 Turbo正式开售打响开门红，9000mAh电池搅局千元市场；长鑫科技IPO明日（5月27日）上会，国产存储龙头即将登陆科创板。618价格战持续，苹果霸榜前三，荣耀、iQOO、红米等安卓机型分食中端市场。荣耀WIN Turbo配置全面曝光，10000mAh电池将于5月29日发布。

---

## 一、存储龙头IPO明日上会：长鑫科技科创板闯关

**2026-05-26** 上交所上市审核委员会定于**明日（5月27日）**召开2026年第27次审议会议，审议长鑫科技集团股份有限公司首发事项。这是科创板试点IPO预先审阅机制后的首单受理项目，备受市场关注。

**核心财务数据：**
- **Q1 2026**：营收508亿元，同比+719%；归母净利润247.62亿元（上年同期亏损15.59亿元），折合日均盈利约2.75亿元
- **上半年预计**：营收1100-1200亿元（+612%至+677%）；归母净利润500-570亿元（+2244%至+2544%）
- **2025年**：扭亏为盈，归母净利润18.75亿元，终结累计超366亿元亏损

**IPO详情：**
- 拟募资：**295亿元**（科创板开板以来第二大IPO，仅次于中芯国际2020年的532.3亿元）
- 保荐机构：中金公司 + 中信建投
- 募资用途：75亿元（晶圆产线升级）+ 130亿元（DRAM技术升级）+ 90亿元（下一代存储前瞻研发）
- 从受理到过会用时仅**148天**，跑出"科创速度"

**市场地位：**
根据Omdia数据，按产能、出货量和销售额统计，长鑫科技已成为**中国第一、全球第四的DRAM厂商**。2025年第四季度全球市场份额已增至**7.67%**。

**点评：** 长鑫科技IPO是国产存储芯片产业的里程碑事件。从连亏十年到日赚近3亿，靠的是存储超级周期红利叠加逆势扩产策略。但上市委现场问询的核心问题是"未来业绩是否存在较大波动风险"——DRAM是强周期行业，当三巨头扩产落地、存储价格回落时，长鑫能否守住市场地位，才是真正的考验。

## 二、vivo Y600 Turbo今日正式开售

**2026-05-27 10:00** vivo Y600 Turbo今日正式开售，Y系列首款带"Turbo"后缀机型，三种配色（沙丘金/电光蓝/千禧粉）。

**售价：**

| 版本 | 首销价 | 国补到手价 |
|------|--------|-----------|
| 8+256GB | 2099元 | 1784元 |
| 12+256GB | 2399元 | 2039元 |
| 12+512GB | 2699元 | 2294元 |

**核心配置：**
- 处理器：第四代骁龙7s（台积电4nm，安兔兔约90万分）
- 电池：**9000mAh蓝海电池**，1200次循环后容量≥80%，**6年耐用**
- 充电：90W闪充 + 全局直驱供电 + 有线反向供电（可为其他设备补电）
- 屏幕：6.83英寸 1.5K OLED直屏，120Hz，局部峰值亮度5000nits，**4320Hz高频PWM调光**
- 散热：7K液冷VC均热板，宣称5年持久流畅
- 影像：5000万索尼LYT-600主摄（支持OIS光学防抖）+ 200万副摄
- 防护：**IP68/IP69防尘防水**，八项军规环境测试，金刚盾玻璃面板
- 其他：全功能NFC、红外遥控、寰宇增强通信3.0、AI网络智选

**用户画像：** 主打"好用耐用、续航在线"，适合外卖员、户外工作者、备用机需求人群。

**点评：** vivo Y600 Turbo以"五年流畅+六年电池耐用"精准命中长周期使用痛点，1784元的国补价极具杀伤力。4320Hz调光和IP69防护下放到2000元档，中端市场的"体验冗余"战正在升级。

## 三、荣耀WIN Turbo配置全面曝光：10000mAh+天玑8500 Elite

**2026-05-25** 荣耀官方宣布WIN Turbo将于**5月29日15:00发布并开售**，这是WIN系列的全新中端机型。

**官方确认配置：**

| 项目 | 规格 |
|------|------|
| 电池 | **10000mAh青海湖电池**，连续游戏14+小时，刷短视频22+小时 |
| 快充 | 80W超级快充 + **27W反向充电**（可化身充电宝） |
| 屏幕 | 6.79英寸OLED绿洲护眼屏，8000nits峰值亮度，**3840Hz零风险调光** |
| 芯片 | 联发科天玑8500 Elite，安兔兔超240万分（同级别最强） |
| 影像 | 5000万主摄，OIS光学防抖，横向大矩阵镜组 |
| 防护 | IP68/IP69/IP69K满级防尘防水 |
| 存储 | 16GB+512GB |
| 配色 | 「快开黑」「指定赢」「不怕蓝」 |
| 通信 | 第二代鸿燕通信，弱网增强（电梯/地下室优化） |
| 其他 | 金属中框，屏下指纹，立体双扬声器，红外，NFC |

**注意：** 本次WIN Turbo**未配备**主动散热风扇（与前两代WIN机型不同）。

**点评：** 荣耀WIN Turbo是继vivo Y600 Turbo之后又一款"续航怪兽"，10000mAh电池在安卓中端机型中极为罕见。8000nits峰值亮度 + 3840Hz调光的护眼配置，瞄准重度游戏用户和户外场景。差异化竞争已从旗舰蔓延到中端，"大电池+强护眼"正成为新标配。
