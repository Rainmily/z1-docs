## 618首周榜单：华为22.68%夺冠苹果19.27%第二

**2026-06-14** 据数码博主"智慧芯片案内人"披露的行业人士数据，**W23周（6/1-6/7）国内手机市场份额排名**正式出炉。**2026年618首周华为以22.68%份额夺冠，同比上升四位，环比持平**。这是国产手机品牌首次在618首周销量上超越苹果。

**2026-06-14** 数码博主"RD观测"透露：截至2026年第23周（6/1-6/7），**华为Mate 80系列累计销量已达约670.56万部**。

按此增速推算，**Mate 80系列年内突破千万销量几无悬念**，将成为继Mate 60系列后的又一"国产街机"。

## REDMI K100系列曝光：10000mAh电池+185Hz超高刷+2亿像素主摄

**2026-06-14** 数码博主爆料，**REDMI K100系列正在测试10000mAh±电池方案**，K系列首次将"超大容量电池"作为核心卖点：

### 核心配置

| 项目 | 配置 |
|------|------|
| **电池** | **10000mAh±** + 100W有线快充 + 可能配50W无线 |
| **屏幕** | 6.6英寸185Hz超高刷OLED直屏+超级像素方案（近1K功耗做2K清晰度） |
| **芯片** | 骁龙8E5（沿用上代旗舰平台，控制成本） |
| **影像** | 2亿像素大底主摄 + 5000万像素潜望长焦 + 长焦微距 |
| **外围** | 3D超声波指纹、金属中框、玻璃后壳、满级防水、对称双扬声器 |

**点评**：K100系列如果10000mAh电池方案落地，将**打破"性能机=大电池+中端芯片"的传统定义**——以往性能机受限于机身厚度和散热，电池多在7000-8000mAh区间。REDMI 10000mAh+骁龙8E5+2亿主摄的组合，瞄准的是"续航党+游戏党+影像党"的全能用户群体。但预计售价会**进一步上探**（同档竞品涨幅更小，K100系列涨价但低于行业均值）。
