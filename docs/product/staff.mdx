# 员工管理

## 产品介绍

员工业绩追踪，提成自动计算，激发团队潜能，提升人效。

## 核心功能

### 权限管理
- 员工账号权限分级
- 操作日志完整记录

### 业绩追踪
- 销售业绩实时统计
- 个人、团队业绩排名

### 提成计算
- 提成规则灵活配置
- 自动计算，透明公正

### 考勤管理
- 打卡签到，考勤记录
- 请假、加班自动统计

### 目标管理
- 目标任务分配到人
- 完成进度实时追踪

### 激励机制
- 业绩排行榜
- 奖励惩罚记录

## 提成方案

| 方案 | 说明 |
|------|------|
| 销售额提成 | 按销售额比例计算 |
| 毛利提成 | 按商品毛利计算 |
| 组合提成 | 多条件组合计算 |
