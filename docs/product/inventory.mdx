# 库存管理

## 产品介绍

从采购入库到销售出库，全流程精细化管理，让库存不再是难题。

## 核心功能

### 多仓库管理
- 支持多仓库统一管理，实时库存查询
- 门店之间可调拨，流程可追溯

### 智能补货
- AI智能分析销售数据，提供补货建议
- 降低缺货风险，优化库存周转

### 串码追溯
- 扫码入库，快速录入
- 串码来源可查，流向可追溯

### 盘点功能
- 支持多种盘点方式，账实相符
- 盘点差异自动计算，一键调整

### 库存预警
- 设置库存上下限，自动预警提醒
- 临保商品提醒，加快周转
