# 智能收银 POS

## 产品介绍

支持多种支付方式，快速开单收银，自动关联会员与库存。

## 核心功能

### 快速开单
- 1秒完成收银，支持扫码快速检索商品
- 支持模糊搜索，输入首字母即可定位商品

### 多渠道支付
- 支持微信、支付宝、现金、银行卡等多种支付方式
- 支持预售、订金、分期付款等特殊场景

### 库存联动
- 自动扣减库存，实时同步多门店数据
- 库存不足自动提醒，防止超卖

### 串码管理
- IMEI串码精准管理，每台手机可追溯
- 销售时自动关联串码，保修可查

### 小票管理
- 支持小票打印，自定义打印模板
- 支持电子小票推送，短信/微信通知

## 产品优势

| 功能 | 说明 |
|------|------|
| 收银速度 | 1秒完成单笔交易 |
| 支付方式 | 微信、支付宝、现金、银行卡、储值卡 |
| 打印支持 | 小票打印机、电子小票 |
| 数据同步 | 多门店实时数据共享 |

## 适用场景

- 手机专卖店
- 数码集合店
- 运营商营业厅
- 连锁手机卖场
