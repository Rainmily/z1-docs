# 产品功能

掌上乾坤提供六大核心模块，覆盖手机连锁门店全场景经营需求。

## 产品模块

- [智能收银 POS](/products/pos) - 快速开单，多种支付方式
- [库存管理](/products/inventory) - 进销存全流程精细化管理
- [会员营销](/products/member) - 精准会员画像，智能营销触达
- [数据报表](/products/report) - 多维度数据分析，洞察经营状况
- [员工管理](/products/staff) - 业效追踪，提成自动计算
- [串码管理](/products/serial) - IMEI全生命周期精准追踪
