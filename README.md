# 掌上乾坤文档中心

掌上乾坤手机连锁门店数智化解决方案官方文档。

## 项目介绍

掌上乾坤专注于手机门店数智化服务10年+，提供从进销存到会员营销，从单店到连锁的全套解决方案。本文档网站基于 [Rspress](https://rspress.dev/) 构建。

## 产品功能

- **智能收银 POS** - 1秒快速开单，支持多种支付方式
- **库存管理** - 进销存全流程精细化管理
- **会员营销** - 精准会员画像，智能营销触达
- **数据报表** - 多维度数据分析，洞察经营状况
- **员工管理** - 业效追踪，提成自动计算
- **串码管理** - IMEI全生命周期精准追踪

## 技术栈

- [Rspress](https://rspress.dev/) - 基于 Rspack 的静态网站生成器
- React - UI 框架
- MDX - 文档格式

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
